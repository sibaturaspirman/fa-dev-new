Perakitan presisi komponen (HDD)


Satu robot menggenggam HDD server dan memasangnya ke dalam server. Sensor gaya memungkinkan perakitan di lokasi dalam yang akan sulit menggunakan hanya sensor penglihatan.

Contoh perakitan server

Solusi untuk Masalah

Masalah

Permintaan untuk otomatisasi yang tidak terpengaruh oleh variasi dalam akurasi pemrosesan komponen (dimensi) dan akurasi perakitan proses hulu semakin meningkat. Untuk perakitan yang memerlukan perakitan presisi sambil melindungi komponen, dalam beberapa kasus perlu menekan dengan gaya tertentu selama jumlah waktu tertentu, dan dalam beberapa kasus perlu mencatatnya.

Solusi

  1. Robot mengeluarkan HDD.
  2. HDD dibawa mendekati slot penyisipan rak.
  3. HDD dimasukkan dalam-dalam ke dalam rak sambil menggunakan sensor gaya dengan cara yang sama seperti tangan manusia merakit.

Point

  • 【Pengurangan kerugian komponen/produk】
  • Sensor gaya mendeteksi dan mencegah kerusakan benda kerja, dll. selama perakitan komponen.
  • 【Kontrol ketertelusuran】
  • Data gaya selama perakitan dikumpulkan secara real time. Data dapat dimanfaatkan untuk pengendalian kualitas.
  • 【Kompensasi pergeseran posisi otomatis】
  • Sensor gaya dan sensor penglihatan mengkompensasi variasi dalam akurasi pemrosesan benda kerja atau posisi yang ditetapkan, berkontribusi pada operasi peralatan yang stabil.
  • 【Perakitan di lokasi dalam】
  • Sensor gaya mencari untuk memungkinkan perakitan bahkan di lokasi yang sulit dideteksi dengan sensor penglihatan.

Contoh perakitan server
