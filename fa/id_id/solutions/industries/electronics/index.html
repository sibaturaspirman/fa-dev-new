Pengencangan sekrup presisi

Dua robot bekerja sama untuk mentransfer benda kerja fleksibel (FPC: sirkuit cetak fleksibel) dan mengencangkan sekrup. Sensor gaya digunakan untuk mendeteksi pergeseran posisi dan kelainan untuk mencegah kerusakan.

Perakitan presisi komponen (memori)

Satu robot melepas dan memasukkan kartu memori. Sensor gaya digunakan untuk merasakan tekanan dan mendeteksi kelainan dan penyelesaian perakitan.

Perakitan presisi komponen (HDD)

Satu robot menggenggam HDD server dan memasangnya ke dalam server. Sensor gaya memungkinkan perakitan di lokasi dalam yang akan sulit menggunakan hanya sensor penglihatan.
