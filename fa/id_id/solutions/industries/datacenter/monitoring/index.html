Solusi : Memperkenalkan SCADA GENESIS64™

GENESIS64™ memberikan visualisasi canggih dengan tampilan 3D dan 2D serta berbagai simbol. Kombinasi fungsi opsional bernilai tambah tinggi dan lingkungan pengembangan dengan tingkat kebebasan tinggi akan berkontribusi pada pembangunan sistem pemantauan yang dibutuhkan pelanggan.

*Untuk model yang tidak dapat dihubungkan ke GENESIS64™, Edgecross dapat digunakan.

Solusi 1
Mendukung penghematan energi melalui visualisasi penggunaan energi dan membantu mengurangi biaya siklus hidup dengan fungsi pemeliharaan preventif peralatan

Tampilan grafis dari data yang dikumpulkan, output formulir, pembuatan alarm, dan analisis mempermudah pemahaman dan peningkatan status operasi fasilitas.
GENESIS64™ juga memastikan ketahanan sistem dengan fitur seperti output log operasi dan pembatasan akun.

Tampilan Grafik

Lihat beberapa tren secara bersamaan untuk mendapatkan gambaran keseluruhan sistem. GENESIS64™ memungkinkan pengguna untuk mengonfirmasi PUE (Power Usage Effectiveness), indikator efisiensi energi pusat data, dengan sekali lihat.

Formulir Hasil

Dari berbagai data yang terakumulasi di server, GENESIS64 dapat secara otomatis menghasilkan formulir untuk berbagai periode waktu (laporan harian/laporan mingguan/laporan bulanan/laporan tahunan, dll.) dan setiap peralatan. Selain Excel®, file dapat dikeluarkan dalam format HTML atau PDF, dan pengguna dapat memilih metode tampilan yang mereka inginkan dari berbagai tabel dan grafik.

Nada alarm

Mengirimkan informasi alarm kepada pengguna melalui email ketika terjadi kesalahan. Dapat juga digunakan untuk peringatan dini, dll. sebelum nilai turun di bawah nilai jaminan kualitas layanan (SLA: Service Level Agreement) dari pusat data.

Pemeliharaan preventif

Merekam/mengakumulasi faktor-faktor kesalahan peralatan untuk memungkinkan pemulihan dini ketika kesalahan berikutnya terjadi.

Solusi 2
Bekerja dengan beberapa sistem sebagai platform untuk pemanfaatan data

GENESIS64™ bekerja dengan alat IT lain dan aplikasi Microsoft®, membantu pelanggan menurunkan total biaya kepemilikan mereka dengan menyediakan platform pemanfaatan data tunggal untuk koordinasi informasi internal dan eksternal. Selain itu, OPC™, MODBUS®, dan BACnet® dapat digunakan sebagai peralatan standar untuk dengan mudah terhubung ke berbagai sistem dan mengurangi biaya koneksi.

*Untuk model yang tidak dapat dihubungkan ke GENESIS64™, Edgecross dapat digunakan.

Solusi 3
Pemantauan/analisis jarak jauh menggunakan terminal pintar memungkinkan pemantauan area luas menggunakan cloud

Pemantauan dan analisis jarak jauh kondisi lapangan dimungkinkan tidak hanya dari komputer pribadi tetapi juga dari terminal pintar seperti smartphone, tablet, dan kacamata pintar. Selain itu, dengan memanfaatkan cloud, dimungkinkan untuk memantau beberapa lokasi sekaligus dan memproses sejumlah besar data.

Mendukung pekerja lapangan

Mengkonsolidasikan data regional di cloud untuk pemantauan area luas
