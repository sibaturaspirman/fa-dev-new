Prediksi waktu kegagalan robot

Fungsi Pemeliharaan Preventif dan Prediktif dengan Kartu Opsi MELFA SmartPlus

Manajemen kesehatan robot melalui pelacakan status

■ Deteksi dini kelainan melalui pemantauan berkelanjutan (Fungsi deteksi kelainan)

Menggunakan teknologi eksklusif untuk mendeteksi tanda-tanda kegagalan sebelum terjadi kegagalan serius. Memungkinkan pemeliharaan dan mencegah penutupan peralatan (Reduction Gear dan Encoder)

■ Periksa tingkat keausan bagian penggerak (fungsi perhitungan keausan).

Tingkat keausan bagian penggerak diperkirakan dari kondisi operasi robot. Pemberitahuan waktu pemeliharaan (output sinyal khusus dan output peringatan), Dukungan untuk pemeliharaan efisien, termasuk prioritas pemeliharaan (pelumas, belt, reduction gear, ball screw, bearing)

■ Simulasi keausan (simulasi pemeliharaan)

Estimasi waktu penggantian suku cadang pemeliharaan dan pemeliharaan bagian yang dioverhaul, siklus pemeliharaan dapat direncanakan sebelumnya. Juga dapat digunakan untuk memverifikasi operasi untuk memperpanjang umur robot

■ Mendukung konstruksi sistem pemeliharaan

Berbagai jenis data pemeliharaan dapat digunakan pada program robot, dan juga dapat dikeluarkan ke sistem IT dan GOT, mendukung pembangunan sistem pemeliharaan yang sesuai untuk peralatan pelanggan.
