Konfigurasi Sistem

fitur

Stabilitas crane dan operasi kecepatan tinggi

Posisi kereta pengangkut dideteksi oleh perangkat pengukur jarak laser, dan kontrol pemosisian dilakukan.

Bahkan jika kondisi mesin berubah, seperti tergelincir atau keausan roda, selalu dapat berhenti di posisi target dengan akurat.

Fungsi anti-ayun meminimalkan ayunan rangka crane saat kereta pengangkut bergerak.

Tidak ada waktu tunggu agar rangka crane berhenti berayun, sehingga waktu takt dipersingkat.

Menurunkan biaya rekayasa

Inverter itu sendiri secara otomatis membuat pola operasi hingga posisi berhenti.

Pemrograman pengontrol tidak diperlukan, dan beban pemrosesan pengontrol dikurangi hanya dengan mengatur kondisi dasar untuk pemosisian di inverter. Dalam kasus kegagalan, pemosisian dapat disesuaikan hanya dengan parameter inverter.

Penghindaran kesalahan crane

Dilengkapi dengan berbagai fungsi keamanan untuk crane penumpuk selain fungsi perlindungan inverter standar

Kontrol keamanan terdistribusi dimungkinkan tanpa hanya mengandalkan fungsi keamanan PLC. Operasi dapat dihentikan dengan aman bahkan jika terjadi gangguan komunikasi.
