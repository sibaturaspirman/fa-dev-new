Network module

Jaringan industri terbuka CC-Link IE memungkinkan konstruksi sistem menggunakan modul Seri MELSEC iQ-R. Jaringan berbasis Ethernet memungkinkan penggunaan kabel dan konektor yang tersedia secara luas. Ini mendukung topologi bintang, garis, dan cincin untuk membangun sistem yang fleksibel. Dalam sistem redundan, jaringan yang sangat andal dengan pengurangan kegagalan titik tunggal menggunakan Jaringan CC-Link IE Field redundan dan modul kepala jarak jauh redundan dapat dikonfigurasi.

Produk terkait

Switch CC-Link IE terkelola NZ2MHG-T8F2

  • Dapat dihubungkan ke perangkat CC-Link IE dan Ethernet secara bersamaan
  • Topologi redundan gaya ERP dan LA antara switch melanjutkan komunikasi saat kegagalan jaringan termasuk pemutusan kabel, dengan mengalihkan jalur jaringan
  • Dengan transceiver SFP*1, kabel serat optik jarak jauh, yang ideal untuk sistem yang memerlukan komunikasi landline fasilitas-ke-fasilitas, tersedia
  • Mendukung VLAN dan dapat mengelola beberapa jaringan dengan satu switch

SFP untuk komunikasi jarak jauh

Dengan fitur transmisi data jarak jauhnya, kabel serat optik ideal untuk komunikasi jarak jauh antar fasilitas

  • *1.Baik port optik (OPT1/OPT2) atau port RJ45 (P1/P2) dapat digunakan pada satu waktu.
