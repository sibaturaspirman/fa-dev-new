Kontrol Proses/Redundan Produk yang Didukung

  • Solusi

Process CPU module, SIL2 process CPU module, Redundant function module (Seri MELSEC iQ-R)

Kontrol proses fleksibel dalam solusi kontrol otomatisasi yang hemat biaya

Sistem kontrol proses MELSEC terdiri dari pengontrol khusus yang dirancang spesifik untuk digunakan dalam otomatisasi proses. CPU sangat fleksibel menggunakan fitur sistem kontrol otomatisasi standar daripada solusi DCS yang bisa mahal untuk diganti dan dipelihara.

  • *1.Jumlah maksimum loop yang dapat digunakan dapat berubah tergantung pada ukuran program aktual yang digunakan. Silakan lihat manual yang relevan untuk detail lebih lanjut.

Sistem redundan lokasi jarak jauh dan pengalihan kecepatan tinggi

Kabel pelacakan serat optik memungkinkan sistem siaga dipasang di lokasi jarak jauh hingga 550 m dari sistem kontrol (primer). Kabel pelacakan kebal terhadap gangguan kebisingan dan mendukung kecepatan transfer data yang cepat.
Kecepatan pengalihan sistem dari sistem kontrol ke sistem siaga juga telah ditingkatkan hingga kecepatan sekitar 10 ms, lebih meningkatkan keandalan sistem.

Penerapan sistem terdistribusi dengan Jaringan CC-Link IE Controller/Field

Dengan menggunakan konverter media, jarak stasiun-ke-stasiun pada Jaringan CC-Link IE Controller dapat diperpanjang hingga 15 km dan jarak stasiun-ke-stasiun pada Jaringan CC-Link IE Field dapat diperpanjang hingga 10 km, memungkinkan konfigurasi sistem terdistribusi.

  • *2.Karena jarak antara sistem redundan terbatas pada 550 m, jarak stasiun-ke-stasiun juga terbatas pada 550 m.
  • *3.Kompatibel dengan konverter media industri yang diproduksi oleh Mitsubishi Electric System & Service Co., Ltd.
    Untuk Jaringan CC-Link IE Controller, silakan lihat DMC-1000SL-DC. Untuk Jaringan CC-Link IE Field, silakan lihat DMC-1000SL-DC atau DMC-1000TS-DC.

Meningkatkan keandalan dengan mengurangi kegagalan titik tunggal

Sistem redundan multi-level dapat direalisasikan dengan memasang sistem kontrol ganda yang terdiri dari CPU kontrol (primer) dan siaga. Dikombinasikan dengan topologi kabel ekstensi ganda untuk unit basis ekstensi redundan dan kabel jaringan dari Jaringan CC-Link IE Field bersama dengan stasiun jarak jauh ganda, risiko kegagalan titik tunggal dapat diminimalkan. Penggantian kabel dan modul secara online (hot-swapping) dimungkinkan sambil terus mengoperasikan sistem ketika terjadi kesalahan, memungkinkan pemecahan masalah yang cepat.

Penambahan stasiun I/O jarak jauh secara online saat sistem berjalan

Dengan memanfaatkan instruksi khusus untuk mengatur parameter modul master/lokal Jaringan CC-Link IE Field, stasiun I/O jarak jauh dapat ditambahkan secara online saat sistem berjalan, meningkatkan ketersediaan sistem.

Konfigurasi campuran modul yang mendukung SIL 2 dan non-pendukung

Sistem redundan SIL2 Seri MELSEC iQ-R memenuhi kebutuhan global dengan mematuhi SIL 2*4, yang diperlukan di bidang infrastruktur publik di mana keandalan tinggi diperlukan.

SIL 2-mendukung input analog

Input analog yang mendukung SIL 2 dikonfigurasikan dengan memiliki total empat modul. Modul ini terdiri dari dua modul input analog (R60AD8-G) dengan isolasi saluran, satu modul output analog (R60DA8-G) dengan isolasi saluran, dan satu modul output digital (RY40PT5B) dengan fungsi diagnostik. Nilai digital yang dihasilkan diverifikasi dengan nilai digital yang dihitung.

SIL 2-mendukung keluaran analog

Output analog yang mendukung SIL 2 dikonfigurasikan untuk memiliki tiga modul secara total. Modul ini terdiri dari satu modul output analog (R60DA8-G) dengan isolasi saluran, satu modul input analog (R60AD8-G) dengan isolasi saluran, dan satu modul output kontrol analog SIL2 (RY40PT5B-AS). Nilai output analog yang dihasilkan diverifikasi dengan nilai yang ditetapkan.

Koordinasi data yang mudah dengan pengontrol terprogram pihak ketiga hanya dengan mendaftarkan parameter

  • Modul CPU proses memungkinkan pertukaran data perangkat dengan pengontrol terprogram pihak ketiga serta pengontrol terprogram Mitsubishi Electric tanpa perlu mengubah program pengontrol terprogram yang ada (fungsi komunikasi CPU sederhana)
  • Fungsi komunikasi CPU sederhana dapat digunakan melalui port Ethernet bawaan dari CPU proses tanpa menggunakan modul antarmuka Ethernet*7
  • *7.Saat digunakan dalam sistem redundan, kegagalan komunikasi pada Ethernet CPU bawaan tidak menjadi faktor pengalihan sistem, tetapi dimungkinkan untuk mengalihkan sistem melalui program.
    Untuk contoh program, silakan lihat Manual Pengguna Modul CPU MELSEC iQ-R (Aplikasi) (SH-081264ENG).
  • *8.Hanya didukung oleh port Ethernet tertanam.

Dengan mudah mendiagnosis dan memantau sistem redundan dari browser web

  • Diagnostik CPU dan pemantauan perangkat*13 apat dilakukan melalui browser web di komputer atau tablet dengan memanfaatkan fungsi server web internal CPU tanpa perlu pembuatan layar oleh pelanggan
  • Diagnosis primer ketika terjadi kesalahan dapat dengan mudah dilakukan tanpa perangkat lunak rekayasa GX Works3
  • Pada layar diagnosis, sistem yang akan dihubungkan dapat dialihkan dari menu
  • Informasi, status kontrol/siaga, dan mode operasi pada sistem (sistem A atau B) dari CPU proses yang saat ini terhubung (mode redundan) dapat diperiksa
  • Dengan menggunakan “User Web Page Design Tool”, alaman web yang disesuaikan dapat dengan mudah dibuat dengan menyusun secara bebas bagian grafis yang disediakan*14
  • *13.File halaman web sistem diperlukan untuk menggunakan fungsi server web modul CPU seri MELSEC iQ-R. Untuk detail tentang cara mendapatkan file, silakan konsultasikan dengan perwakilan Mitsubishi setempat.
  • *14.Untuk detail tentang cara mendapatkan "User Web Page Design Tool", silakan berkonsultasi dengan perwakilan Mitsubishi setempat.
