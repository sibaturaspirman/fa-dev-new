SolusiPerawatan Menyeluruh

Dalam lingkungan bisnis yang berubah drastis saat ini, dampak penghentian peralatan secara tiba-tiba terhadap laba perusahaan sangat besar, dan semakin banyak bisnis yang menerapkan pemeliharaan peralatan terencana dengan tujuan mencapai pabrik tanpa henti.
Sementara itu, industri manufaktur menghadapi masalah besar lainnya dalam mewariskan keahlian karyawan yang sangat berpengalaman.

Kasus12Memulihkan program perangkat FA di tempat segera

Perawatan
Korektif

Tingkat peralatan

Masalah

Jika terjadi kegagalan PLC atau baterai mati, peralatan baru dan komputer untuk menulis program harus disiapkan, yang berarti pemulihan membutuhkan waktu lama.

Solusi

Aktifkan pencadangan/pemulihan program dan parameter perangkat FA dengan mudah

Fungsi Pencadangan/Pemulihan GOT memungkinkan pengguna untuk mencadangkan program dan parameter perangkat FA seperti PLC ke kartu memori SD GOT ​​atau memori USB. Program dan parameter dapat dengan mudah dipulihkan dari GOT ke perangkat FA saat perangkat perlu diganti.

Keuntungan

Dengan melakukan pencadangan terlebih dahulu, perangkat FA dapat dipulihkan dari GOT di lokasi tanpa PC jika terjadi kegagalan perangkat FA, yang mengurangi waktu henti.
