  • Katalog Produk

    Merampingkan Proses Startup

    Salin parameter dengan perangkat memori USB

    Konektor host USB (tipe A), yang memungkinkan koneksi perangkat eksternal, telah ditambahkan.
    Parameter dapat disalin ke perangkat memori USB komersial.

    Pengaturan mudah dengan FR Configurator2

    • Dengan rasa kesatuan dengan produk Mitsubishi FA lainnya dengan desain dan pengoperasian MELSOFT yang sama, perangkat lunak ini mudah digunakan.
    • Koneksi plug-and-play yang mudah tersedia pada terminal USB yang dilengkapi sebagai standar.
    • Versi uji coba gratis, yang berisi fungsi-fungsi awal, tersedia. Versi ini dapat diunduh di Situs Web Global Mitsubishi Electric FA.

    Pengkabelan mudah ke sirkuit kontrol

    Terminal penjepit pegas telah diadopsi untuk terminal sirkuit kontrol.
    Kabel dapat dilindungi dari kendur akibat getaran selama pengangkutan inverter. Sepuluh terminal tambahan digunakan dibandingkan dengan seri FR-F700(P).
    Terminal penjepit bundar juga dapat digunakan dengan menggunakan opsi terminal kontrol (FR-A8TR).

    Tampilan yang Mudah Dipahami Meningkatkan Kemampuan Operasional

    Pengoperasian yang Mudah dengan GOT

    • Komunikasi otomatis dapat dilakukan tanpa menentukan pengaturan parameter apa pun, cukup dengan menghubungkannya ke seri GOT2000.
    • Monitor perangkat fungsi PLC dapat ditampilkan di seri GOT2000. Kontrol batch beberapa monitor perangkat inverter dapat dilakukan dengan satu unit GOT.
    • Contoh data layar untuk FR-F800 dapat ditemukan di perangkat lunak desain layar seri GOT2000. Untuk versi terbaru perangkat lunak desain layar, silakan hubungi kantor penjualan setempat.

    Konfigurasi parameter yang mudah diikuti

    Dengan pemilihan mode pengaturan parameter pada panel operasi, mode parameter grup dapat dipilih untuk menyediakan pengaturan parameter yang intuitif dan sederhana. (Mode pengaturan parameter konvensional dipilih secara default.)

    Panel operasi yang mudah dibaca

    Tampilan 5 digit, 12 segmen telah diadopsi untuk panel operasi (FR-DU08) untuk tampilan karakter yang lebih alami. Selain itu, panel operasi opsional (FR-LU08) yang mengadopsi panel LCD yang mampu menampilkan teks dan menu juga tersedia.

    Untuk Membantu Perawatan

    Mengurangi waktu pemeriksaan kabel

    Penutup tipe terpisah disesuaikan untuk semua model kapasitas. Perawatan kini mudah karena yang perlu dilakukan operator hanyalah melepas penutup untuk area kabel yang dituju.

    Perawatan dan kontrol beberapa inverter (Opsi)

    Pembacaan nomor seri dapat dilakukan menggunakan panel operasi LCD opsional (FR-LU08) atau perangkat lunak pengaturan inverter (FR Configurator2). Administrasi berbagai inverter menjadi jauh lebih mudah.
