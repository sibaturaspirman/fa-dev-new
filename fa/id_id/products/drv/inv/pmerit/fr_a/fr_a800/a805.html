    FR-A800-GN

    Dengan fungsi komunikasi CC-Link IE TSN (Time Sensitive Networking), data dapat dikirimkan ke sistem TI sambil menjalankan kontrol komunikasi siklik secara real-time. Ini akan berkontribusi pada pengurangan waktu mulai dan peningkatan kemudahan perawatan.

    Pengurangan waktu mulai

    Nomor stasiun dapat diatur dengan mudah menggunakan sakelar putar.
    Deteksi otomatis konfigurasi jaringan oleh Engineering Software (GX Works3) mengurangi waktu pengaktifan.
    Masalah saat pengaktifan seperti gangguan jaringan dapat ditemukan sekilas dengan fungsi diagnostik.

    Peningkatan pemeliharaan

    Sinkronisasi waktu memungkinkan pemantauan secara real-time.
    Hal ini memungkinkan analisis masalah dilakukan segera setelah kesalahan terjadi.
    FR Configurator2 dapat dihubungkan melalui Ethernet, yang memudahkan pekerjaan pemeliharaan.

    FR-A800-GF

    Inverter memiliki fungsi komunikasi CC-Link IE Field Network, yang memungkinkan pengoperasian langsung melalui CC-Link IE Field Network.

    FR-A800-E

    CC-Link IE Field Network Basic didukung, sehingga jaringan dapat dibuat dengan mudah. ​​Status inverter dapat dipantau dan parameter dapat diatur melalui Internet. (MODBUS/TCP juga didukung.)

    Komunikasi jaringan lainnya

    Opsi komunikasi

    • CC-Link, SSCNET III (/H), DeviceNet™, PROFIBUS-DPV0 didukung menggunakan opsi komunikasi yang kompatibel. Komunikasi berbasis Ethernet lainnya seperti komunikasi CC-Link IE Field Network dan komunikasi jarak jauh FL juga dapat didukung.
    • Pemrograman blok fungsi (FB) untuk komunikasi CC-Link tersedia untuk seri MELSEC-Q/L guna membuat program urutan kontrol inverter dengan mudah. (Perpustakaan FB (kumpulan elemen FB) dapat diunduh dari Situs Web Global Mitsubishi Electric FA.)
    • Model standar dengan antarmuka RS-485 (protokol inverter Mitsubishi, protokol MODBUS® RTU) memungkinkan komunikasi dengan perangkat lain tanpa menggunakan opsi komunikasi.
