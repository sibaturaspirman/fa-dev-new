Memulai dan menyesuaikan dengan mudah 

Parameter dapat digunakan untuk penyesuaian mekanis sesuai dengan aplikasi, berguna untuk pekerjaan permulaan dan penyesuaian sistem.

Sebelum

Pengaturan dan penyesuaian beberapa perangkat termasuk pengontrol diperlukan untuk mengendalikan penari, dan butuh banyak waktu untuk memulai sistem.

Sebelum

Ada kekhawatiran tentang kompatibilitas dengan sistem yang ada.

Setelah

  • Kontrol posisi yang rumit dari gulungan penari dapat dicapai hanya dengan inverter dengan mengatur parameter.
  • Dengan mengatur spesifikasi mekanis, kontrol yang optimal dapat dilakukan sesuai dengan sistem dan aplikasi.
  • Metode input sinyal analog/pulsa dapat dipilih sesuai dengan kebijakan pelanggan. Input melalui komunikasi juga tersedia.
  • Kontrol PID memungkinkan dan menyederhanakan kontrol yang rumit hanya dengan menggunakan inverter.
  • Penyesuaian penguatan PI tegangan otomatis memungkinkan pengaktifan yang mudah. ​​(Penyetelan penguatan PI tegangan)

Contoh prosedur memulai

Prosedur berikut menunjukkan contoh pengaturan parameter untuk kontrol kecepatan umpan balik penari.

Lakukan pengaturan sesuai dengan jenis motor dan metode kontrol.

Pengaturan parameter dasar dan pemilihan metode kontrol

Tetapkan nilai untuk setiap parameter sesuai dengan metode kontrol dan jenis motor.
(Penyesuaian penguatan kontrol kecepatan atau penyetelan otomatis offline diperlukan sesuai dengan metode kontrol.)

  • *1Untuk metode kontrol, kontrol vektor direkomendasikan.
  • *2Pengaturan diperlukan untuk motor selain motor Mitsubishi Electric
    (motor seri SF-PR, SF-JR, SF-HR, SF-JRCA, SF-HRCA, atau SF-V5RU (1500 putaran/menit)).

Tetapkan spesifikasi mekanis.

Pengaturan spesifikasi mekanis

Tetapkan spesifikasi mekanis sesuai dengan aplikasi.

Kontrol peningkatan akurasi dengan perhitungan diameter lilitan

Dengan menghitung diameter lilitan poros lilitan/pelepasan, tegangan selalu dioptimalkan meskipun berubah seiring dengan perubahan diameter lilitan.

Pilih metode input dan fungsi terminal input untuk perintah kecepatan baris.

Pemilihan metode input untuk perintah kecepatan baris, sinyal penari, dan kecepatan baris aktual

Metode input perintah kecepatan baris dapat dipilih dari berikut ini: input analog melalui terminal (2, 4, 1, 6, dst.), input rangkaian pulsa fase tunggal, input pulsa encoder, dan input melalui komunikasi (komunikasi CC-Link IE Field Network, DeviceNet™, PROFIBUS-DPV0, dst.).

Tetapkan parameter untuk mengendalikan gerakan penari dan menyesuaikan penguatan ketegangan PI.

Pengaturan posisi target gerakan guling penari

Tetapkan posisi target, batas atas, dan batas bawah untuk gerakan guling penari.

Penyesuaian penguatan PI secara otomatis

Penguatan PI secara otomatis disesuaikan dengan penyetelan penguatan PI tegangan. Waktu yang diperlukan untuk penyesuaian penguatan dapat dikurangi.
