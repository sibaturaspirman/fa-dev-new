Dukungan sistem

Kompatibilitas dengan berbagai jaringan terbuka

Dengan menggunakan pengontrol, inverter dapat dikontrol dan dipantau melalui berbagai jenis jaringan.

Komunikasi EthernetFR-A800-E-CRN

(1) CC-Link IE Field Network Basic didukung

CC-Link IE Field Network Basic didukung, sehingga jaringan dapat dibuat dengan mudah. Status inverter dapat dipantau dan parameternya dapat diatur melalui Internet. (MODBUS/TCP juga didukung.)

(2) Pemantauan dari lokasi jarak jauh

Inverter FR-A800-E-CRN dapat dihubungkan ke FR Configurator2 menggunakan titik akses LAN nirkabel industri*1 yang tersedia secara komersial.*2 Penyesuaian parameter inverter, pemantauan inverter (pemantauan simultan beberapa sumbu dimungkinkan) dan perawatan inverter seperti pemeriksaan diagnosis masa pakai dapat dilakukan secara nirkabel.

  • *1LAN nirkabel yang cocok untuk penggunaan industri di lingkungan yang keras atau di lingkungan yang membutuhkan keandalan tinggi (redundansi).
  • *2Dalam lingkungan atau kondisi instalasi tertentu, komunikasi Ethernet melalui LAN nirkabel tidak stabil seperti komunikasi melalui LAN kabel. Sebelum memulai pengoperasian, selalu periksa status komunikasi. Untuk aplikasi yang membutuhkan transmisi data atau pembaruan secara berkala atau dalam jangka waktu tertentu, koneksi kabel direkomendasikan.

Komunikasi jaringan lainnya

  • CC-Link, SSCNETIII (/H), DeviceNet™, PROFIBUS-DPV0 didukung menggunakan opsi komunikasi yang kompatibel. Komunikasi berbasis Ethernet lainnya seperti komunikasi CC-Link IE Field Network dan komunikasi jarak jauh FL juga dapat didukung.
  • Pemrograman blok fungsi (FB) untuk komunikasi CC-Link tersedia untuk seri MELSEC-Q/L guna membuat program urutan kontrol inverter dengan mudah. (Perpustakaan FB (kumpulan elemen FB) dapat diunduh dari Situs Web Global Mitsubishi Electric FA.)
  • Model standar dengan antarmuka RS-485 (protokol inverter Mitsubishi, protokol MODBUS® RTU) memungkinkan komunikasi dengan perangkat lain tanpa menggunakan opsi komunikasi.
