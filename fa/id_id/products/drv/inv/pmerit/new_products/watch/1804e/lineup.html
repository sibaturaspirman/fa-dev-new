Opsi deteksi fase

FR-A8AVP mengubah inverter FR-A842 menjadi konverter faktor daya tinggi. Jika dikombinasikan dengan FR-A8VPB-H, FR-A8AVP dapat mengirimkan informasi fase dan tegangan tentang catu daya komersial ke inverter yang terhubung ke konverter FR-A842. Tidak mungkin menggunakan opsi plug-in lain saat FR-A8AVP dipasang pada konverter FR-A842. Semua opsi terminal juga tidak dapat digunakan.

Kotak transformator pendeteksi fase

Ini adalah transformator stepdown yang memperoleh informasi fase dan tegangan tentang sistem daya untuk dikirim ke konverter FR-A842.

Kapasitor filter khusus

Ini adalah kapasitor filter yang dibuat khusus untuk konverter FR-A842 guna meningkatkan faktor daya input dan mengurangi harmonik dalam catu daya.

Reaktor filter khusus

Ini adalah reaktor filter yang dibuat khusus untuk konverter FR-A842 guna meningkatkan faktor daya input dan mengurangi harmonik pada catu daya.

Reaktor khusus untuk kontrol PWM

Ini adalah reaktor kontrol PWM yang dibuat khusus untuk konverter FR-A842.

Komponen sirkuit khusus untuk proteksi arus masuk
