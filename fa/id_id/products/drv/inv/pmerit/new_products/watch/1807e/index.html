Inverter FR-A800-E-R2R (Model Komunikasi Ethernet dari inverter khusus roll to roll)

Fitur

Fungsi (protokol) berikut tersedia melalui komunikasi Ethernet untuk keperluan umum.

CC-Link IE Field Network Basic
Koneksi produk MELSOFT/FA
SLMP*1
MODBUS/TCP

*1：SLMP adalah protokol umum untuk komunikasi yang lancar antar aplikasi. Pengguna tidak perlu khawatir dengan lapisan atau batasan jaringan.

*2：Koneksi produk MELSOFT / FA, SLMP, dll.

Manfaat fungsi Ethernet

Pemantauan jarak jauh

Dengan menghubungkan inverter FR-A800-E-R2R dan FR Congurator2* ke Ethernet, keduanya dapat berkomunikasi satu sama lain (* FR Congurator2 akan segera ditingkatkan untuk mendukung inverter FR-A800-E-R2R). Di FR Congurator2, data dari gulungan penari dan sensor ketegangan yang dikirim ke inverter FR-A800-E-R2R dapat ditampilkan dalam bentuk grafik.
Pemantauan data dan pengaturan parameter untuk kontrol ketegangan dimungkinkan di FR Congurator2 melalui jaringan. Waktu mulai dan waktu penyesuaian sistem rol ke rol dapat dikurangi.
