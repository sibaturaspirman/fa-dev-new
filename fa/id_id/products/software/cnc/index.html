Daftar Produk

NC Servo Selection

Membantu dalam pemilihan kapasitas servo.

NC Designer2

Membantu dalam pembuatan layar di Windows.

NC Trainer2/NC Trainer2 plus

Perangkat lunak pelatihan Mitsubishi CNC (NC Trainer2) dan perangkat lunak dukungan pengembangan khusus (NC Trainer2 plus).
Berbagai macam alat pengembangan yang membantu dalam individualisasi CNC.

NC Configurator2

Membantu dalam mengonfigurasi parameter untuk beberapa CNC yang terhubung ke jaringan.

NC Analyzer2

Membantu dalam penyesuaian parameter servo.

NC Explorer

Memungkinkan data program pemesinan ditransfer dari NC dan komputer host melalui koneksi Ethernet.

NC Monitor2

Memungkinkan beberapa CNC dipantau dan dikontrol dari jarak jauh pada saat yang sama.

Remote Monitor Tool (C70)

Memungkinkan NC dipantau dan dikontrol dari komputer.

NC Virtual Simulator

Simulator fidelitas tinggi mendigitalkan proses dari perancangan produk hingga pemesinan uji, meningkatkan efisiensi kerja dan produktivitas.

NC Machine Tool Optimizer

Perangkat lunak ini menganalisis dan mendiagnosis status operasi yang dikumpulkan dari peralatan mesin dan peralatan periferal.
