Konverter file untuk EcoAdviser

Perangkat lunak ini adalah alat yang mengonversi file pencatatan Unit Pencatatan, Perangkat Lunak Akuisisi Data, atau MELSEC ke dalam format yang dapat dikumpulkan oleh EcoAdviser.
File yang dikonversi dapat dikumpulkan secara manual dengan EcoAdviser dan digunakan untuk grafik dan laporan.
Konverter file untuk EcoAdviser

Konversi data

  1. ①Unit Pencatatan
    Data yang dicatat oleh Unit Pencatatan diimpor dari kartu memori SD ke komputer, dan perangkat lunak ini mengubahnya ke dalam format yang dapat dikumpulkan oleh EcoAdviser.
  2. ②Perangkat Lunak Akuisisi Data
    Perangkat lunak ini mengubah data yang dicatat oleh Perangkat Lunak Akuisisi Data ke dalam format yang dapat dikumpulkan oleh EcoAdviser.
  3. ③MELSEC
    Data yang dicatat oleh MELSEC diimpor ke komputer, dan perangkat lunak ini mengubahnya ke dalam format yang dapat dikumpulkan oleh EcoAdviser.
    * Silakan rujuk manual pengguna MELSEC untuk mengetahui cara mengimpor berkas pencatatan MELSEC ke komputer Anda.

Pengonversi berkas untuk EcoAdviser (Versi GENESIS64)

Pengonversi berkas untuk mengonversi data yang dikumpulkan oleh GENESIS64 ke dalam format yang dapat dikumpulkan oleh EcoAdviser.
Data dari GENESIS64 dapat digunakan untuk diagnosis dengan EcoAdviser.
Silakan hubungi perwakilan penjualan setempat untuk mengetahui detail selengkapnya.
