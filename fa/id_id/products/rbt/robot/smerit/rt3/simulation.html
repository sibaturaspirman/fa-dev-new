Robot engineering software (MELFA)MELSOFT RT ToolBox3

  • Fitur Produk

Fungsi penyuntingan dan debugging program

  • Program dapat disunting menggunakan MELFA-BASIC IV, V, dan VI. (bervariasi tergantung pada modelnya).
  • Peningkatan operasi kerja dengan format multi-jendela dan berbagai fungsi penyuntingan.
  • Ini berguna untuk digunakan dalam operasi pemeriksaan seperti pelaksanaan langkah-langkah program, pengaturan pengaturan breakpoint, dan tugas-tugas lainnya.

MELFA-BASIC VI

Selain menyediakan serangkaian perintah yang lebih lengkap, ini menggunakan pemrograman terstruktur untuk memberikan tingkat penggunaan ulang dan keterbacaan yang tinggi.

  • Memungkinkan program terstruktur, yang memungkinkan pemrograman dengan tingkat penggunaan ulang dan keterbacaan yang tinggi.
  • Menyimpan pustaka pemrosesan program memungkinkan pengetahuan untuk diakumulasikan dan memberikan penggunaan ulang yang lebih baik.
    Pustaka juga dapat disembunyikan untuk mencegah pengetahuan diungkapkan.

3D viewer

  • Penampil 3D memungkinkan verifikasi pose dan gerakan robot dengan mudah, verifikasi nilai batas parameter yang ditentukan pengguna, dan penempatan virtual perangkat periferal berdasarkan objek dasar.
  • Penampil ini juga dapat digunakan untuk memeriksa interferensi antara robot dan perangkat periferal.

Fungsi simulasi

Program robot, termasuk sinyal I/O, dapat disimulasikan di PC, gerakan dapat diverifikasi, dan waktu siklus operasi robot juga dapat diukur.
Artinya, operasi sistem yang sebenarnya dapat direproduksi sebagaimana adanya. Fungsi simulasi tersebut juga efektif untuk pemeriksaan awal sistem.

  • Maksimal 8 robot dapat dioperasikan, dan gerakan terkoordinasi antar robot dapat diverifikasi.
  • Simulasi servo juga dapat dilakukan, untuk pemeriksaan awal beban.
  • Sinyal dapat dikoordinasikan dengan GX works2 dan GX works3 untuk pembuatan simulator garis yang mudah.

Atas halaman

Fungsi MELFA-Works

Fungsi MELFA Works dapat digunakan di RT ToolBox3 Pro.
MELFA Works, alat tambahan SolidWorks®, dapat mensimulasikan sistem produksi robot di SolidWorks® dan mengeluarkan data jalur pemrosesan pada benda kerja.*1)
*1) SolidWorks® adalah merek dagang terdaftar dari SolidWorks Corp, (AS). Alat tambahan adalah program perangkat lunak yang menambahkan fungsi tertentu ke paket perangkat lunak aplikasi.

Fungsi utama

  • Simulasi operasi robot
    Program robot, termasuk sinyal I/O, dapat disimulasikan di SolidWorks®. Artinya, operasi sistem aktual dapat direproduksi sebagaimana adanya.
  • Waktu siklus
    Waktu siklus operasi robot dapat diukur.
  • Pemeriksaan interferensi
    Interferensi antara robot dan perangkat periferal dapat diperiksa.
  • Tautan CAD
    Data kerja untuk melakukan operasi penyegelan dan tugas lain yang memerlukan banyak langkah pengajaran dapat dibuat dengan mudah dengan memilih area pemrosesan pada data CAD 3D. Karena data kerja dibuat dari data CAD 3D, kurva 3D yang rumit pun dapat dibuat.
  • Kalibrasi
    Data urutan titik koordinat CAD yang dibuat menggunakan tautan CAD dikoreksi menjadi data koordinat robot, dan program operasi serta data urutan titik dikirim ke robot.
