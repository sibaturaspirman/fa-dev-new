Intelligent Options
MELFA Smart Plus

  • Katalog Produk

Ini adalah opsi yang menghadirkan kecerdasan generasi berikutnya pada robot industri MELFA.

Fungsi cerdas tingkat lanjut menyediakan fungsi terdepan untuk semua fase operasi pelanggan, mulai dari desain dan permulaan hingga operasi dan pemeliharaan.

Selengkapnya

  1. 1.Ini adalah fungsi yang mendukung produksi sel, menggunakan robot untuk mengatasi keterbatasan waktu tunggu, volume produksi, dan lokasi.
  2. 2.Seperti fungsi integrasi untuk berbagai sensor dan fungsi penyesuaian startup otonom.
  3. 3.Memanfaatkan teknologi AI milik Mitsubishi Electric (Maisart).

Daftar Produk

Predictive maintenance function

Komponen penggerak yang rusak terdeteksi sebelum ketidaknormalan pada perilaku robot terlihat. Waktu henti peralatan produksi berkurang.

Preventive maintenance function

Pelacakan status pengoperasian robot membantu mengelola kondisi robot. Perawatan kini menjadi lebih efisien.

Enhancement function for force sense control

Parameter untuk pola operasi optimal ditemukan melalui pelatihan dalam waktu singkat. Waktu pengaturan dan waktu sentuh sensor gaya berkurang.

MELFA-3D Vision enhancement function

Waktu mulai yang lebih singkat untuk visi MELFA-3D berkat penyesuaian parameter otomatis yang memanfaatkan teknologi AI milik kami "Maisart".

2D vision sensor enhancement function

Sistem robot dapat diatur dengan mudah menggunakan sensor penglihatan tanpa pengetahuan khusus.

Calibration assistance function

Pengaturan sensor penglihatan 2D yang mudah dan peningkatan presisi pekerjaan. Pengajaran yang memakan waktu diotomatisasi sehingga meningkatkan akurasi.

Coordinated control of additional axis

Pengoperasian yang terkoordinasi antara robot dan sumbu tambahan memungkinkan robot bekerja pada benda kerja yang melampaui jangkauan operasinya.

Robot mechanism thermal compensation function

Mengimbangi ekspansi termal lengan robot untuk meningkatkan akurasi posisi.
