Pengenalan seri RH-CRH

Lengan yang ringkas untuk menghemat ruang dan lengan robot yang ringan.

Muatan maksimum: 3kg, tipe 6kg

  • Model: RH-3CRH/RH-6CRH
  • Tinggi keseluruhan:
    RH-3CRH : 500mm/RH-6CRH : 520mm
  • Berat robot:
    RH-3CRH : 14kg
    RH-6CRH6020 : 17kg
    RH-6CRH7020 : 18kg

Muatan maksimum: 10kg, tipe 20kg

  • Model: RH-10CRH/RH-20CRH
  • Tinggi keseluruhan:
    RH-10CRH : 560mm/RH-20CRH : 774mm
  • Berat robot:
    RH-10CRH60XX/70XX*1 : 20kg
    RH-10CRH80XX*1 : 21kg
    RH-20CRH8542 : 54kg
    RH-20CRH10042 : 57kg
    (*1 : XX= 20 or 30)

Atas halaman

Fitur

Mengurangi tinggi keseluruhan untuk desain yang ringkas
  • Dapat dipasang di ruang terbatas.
  • Dapat dimasukkan ke dalam berbagai macam sistem.
  • Pemasangan mudah karena sederhana dan ringan.
Pengoperasian berkecepatan tinggi dan kinerja tinggi untuk produktivitas.
  • CC-Link IE Field Basic sebagai standar.
  • Produktivitas ditingkatkan karena peningkatan kinerja operasi berkelanjutan.
  • Fungsi cerdas yang didukung oleh Smart Plus. (*2)
    （*2: Tidak didukung untuk RH-10CRH/20CRH）
Motor tanpa baterai (RH-10CRH/20CRH)
  • Menghilangkan kebutuhan untuk penggantian baterai secara berkala
  • Tidak ada risiko kehilangan origin karena baterai lemah
