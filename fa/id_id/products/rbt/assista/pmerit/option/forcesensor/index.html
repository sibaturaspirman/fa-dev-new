Opsi Cerdas Force sensor set

  • Fitur Produk

Perkenalan

Tugas perakitan/pemrosesan dilakukan dengan cara yang sama seperti manusia, sambil merasakan gaya yang diterapkan pada gripper.

  • Mengendalikan robot agar bergerak dengan hati-hati di sepanjang kontur benda kerja.
  • Beroperasi dengan gaya konstan ke arah yang ditentukan oleh pengguna.
  • Mengubah tingkat kehalusan untuk gerakan robot dan kondisi deteksi kontak selama pengoperasian.
  • Mendapatkan data posisi dan gaya pada saat kontak.

Silakan lihat "Katalog Seri MELFA FR" untuk detail "Force sensor set".

Atas halaman

Fitur

Meningkatkan stabilitas produksi

  • Mengontrol “kekuatan” dan “fleksibilitas”. Memodifikasi properti kontrol selama pengoperasian.
  • Menjaga gaya tetap konstan sehingga benda kerja dapat ditangani tanpa menyebabkan kerusakan.
  • Kualitas dapat dikelola menggunakan data log, dan penyebab kesalahan kerja dapat dianalisis.

Realisasi tugas perakitan dan pemrosesan yang kompleks

  • Mengganti operasi sebagai respons terhadap status transisi.
  • Tugas perakitan yang kompleks dicapai melalui teknik seperti pencocokan fase.

Kontrol Mudah dan Operasi Sederhana

  • Berdasarkan contoh program, program dapat dibuat dengan mudah menggunakan bahasa robot khusus.
  • Robot dapat dengan cepat [diajarkan] posisi yang akurat berdasarkan data posisi dan gaya dari kotak pengajaran. Kondisi kerja dapat diverifikasi dan disesuaikan dengan melihat posisi, data gaya, dan bentuk gelombang grafik.

Contoh Fungsi

Pengenalan beberapa fungsi.

Fungsi dasar


Kontrol gaya
Mengontrol “gaya” dan “fleksibilitas”. Memodifikasi properti kontrol selama pengoperasian.
Menjaga gaya tetap konstan sehingga benda kerja dapat ditangani tanpa menyebabkan kerusakan.

Deteksi gaya
Mengalihkan operasi sebagai respons terhadap status transisi.
Tugas perakitan yang rumit dicapai melalui teknik seperti pencocokan fase.

Log tenaga
Memeriksa status kerja untuk memudahkan penyesuaian. Analisis data log juga memungkinkan langkah-langkah keselamatan prediktif.

Fungsi Pembatalan Offset Gravitasi

Mengompensasi gravitasi sebagai respons terhadap perubahan gaya pada sensor gaya di arah X, Y, dan Z saat sudut berubah. Kontrol gaya dapat dilakukan dengan benar bahkan saat sudut berubah.

Bantuan pekerjaan mengajar

GUI gaya disertakan (*1)
(*1:Antarmuka Pengguna Grafis)

  • Perangkat lunak komputer (RT ToolBox3) dan kotak pengajaran (R86TB atau R32TB) adalah fitur standar layar GUI gaya, yang memudahkan penggunaan sensor gaya.
  • Pengajaran dapat dilakukan sambil memantau gaya reaktif pada layar GUI gaya.
  • Pengaturan logika keselamatan, pemantauan sinyal, variabel, dan kondisi beban, serta lebih banyak fitur dapat digunakan hanya dengan satu kotak pengajaran ini.
  • Data gaya yang disinkronkan dengan data posisi dapat disimpan sebagai data log.
  • Data log dapat dilihat sebagai grafik menggunakan RT ToolBox. Selain itu, berkas data log dapat diunduh ke komputer melalui FTP.
