Pengenalan MELSOFT RT VisualBox

Tidak Memerlukan Keahlian untuk Robot.

  • Anda dapat membuat program secara visual menggunakan operasi akuarium dengan RT-VisualBox.
  • Perangkat lunak ini memungkinkan operator untuk memprogram robot ini dengan antarmuka pemrograman "train by demonstrasi".
  • Hal ini memungkinkan mereka untuk menggerakkan posisi lengan robot dan mengatur titik-titik arah dengan mudah.

Layar utama RT VisualBox

Gambar di bawah ini menjelaskan antarmuka layar utama.

Atas halaman

Mode Pemrograman/3D

Mengetuk/mengklik tombol mode Pemrograman/3D memungkinkan Anda untuk beralih antara mode Pemrograman dan mode 3D.
Pengoperasian Layar: Pengoperasian Sentuh dan Pengoperasian Mouse yang Setara.
(Layar tidak dapat digeser, digeser, dan diputar menggunakan mouse.)

Area 3D

Menampilkan model 3D robot dan posisi yang telah diajarkan kepadanya.
Dalam mode simulasi, robot di area 3D di layar akan bergerak sesuai dengan program yang dibuat.
Tindakan: memutar tampilan di sekitar robot.

Atas halaman

Mode pemrograman

Area yang digunakan untuk membuat program dengan menempatkan dan mengatur blok program
Tindakan: menambah/mengurangi ukuran program/memindahkan blok program ke atas, ke bawah, ke kiri, dan ke kanan.

Pemrograman Visual

Anda dapat membuat program secara visual menggunakan operasi intuitif dengan RT VisualBox.
  • Program dapat dibuat dengan menyeret dan meletakkan blok program.
  • Program dapat dijalankan dan di-debug.
  • Tata letak dan simulasi 3D dapat dilihat dari layar ini.
