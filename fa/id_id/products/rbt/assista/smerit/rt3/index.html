Pengenalan MELSOFT RT ToolBox3

Perangkat lunak pembuatan program dan dukungan rekayasa total.

Ini adalah perangkat lunak komputer untuk membantu berbagai tugas mulai dari memulai sistem hingga debugging dan pengoperasian.
Lini produk meliputi [RT ToolBox 3 Pro] yang memungkinkan simulasi robot dijalankan pada perangkat lunak CAD 3D.

  • Fungsi utama
    Membuat dan mengedit program
    Memeriksa lingkungan operasi sebelum pemasangan robot
    Simulasi operasi robot di komputer
    Memperkirakan waktu siklus
    Men-debug saat robot dinyalakan
    Memantau status dan kesalahan robot selama operasi

RT ToolBox3 Pro

Fungsi MELFA Works dapat digunakan di RT ToolBox3 Pro.
MELFA Works, alat tambahan SolidWorks®, dapat mensimulasikan sistem produksi robot di SolidWorks® dan mengeluarkan data jalur pemrosesan pada benda kerja.*1)
*1) SolidWorks® adalah merek dagang terdaftar dari SolidWorks Corp, (AS). Alat tambahan adalah program perangkat lunak yang menambahkan fungsi tertentu ke paket perangkat lunak aplikasi.
