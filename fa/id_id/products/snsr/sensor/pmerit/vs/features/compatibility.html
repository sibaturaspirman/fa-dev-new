Vision Sensor Fitur Kompatibilitas

  • Product information

Kompatibilitas dengan produk otomasi pabrik Mitsubishi Electric

  • Sensor penglihatan kompatibel dengan CC-Link IE Field Network Basic*1, iQSS, dan SLMP, yang memungkinkan koneksi yang lebih mudah ke produk Mitsubishi Electric seperti pengontrol yang dapat diprogram, robot, dan pengontrol gerakan
  • Pengaturan kontrol/pemantauan/parameter sensor penglihatan dan akuisisi hasil dapat dilakukan dari pengontrol yang dapat diprogram
    (Misalnya) Menyalakan/mematikan perangkat dapat memicu sensor penglihatan
  • Gambar pengenalan sensor penglihatan ditransfer ke GOT (HMI) atau sistem IT melalui server FTP, mewujudkan ketertelusuran dengan membandingkan hasil pengenalan dan gambar pengenalan

Kompatibel dengan CC-Link IE Field Network Basic

CC-Link IE Field Network Basic fungsi antarmuka tertanam, mewujudkan kompatibilitas tinggi dengan pengontrol yang dapat diprogram seperti pengaturan yang lebih mudah. ​​Nilai terukur dan hasil penilaian diperoleh melalui jaringan. Selain itu, pengaturan sensor penglihatan dapat diperiksa dan diubah dari pengontrol yang dapat diprogram.
