Kompatibel dengan produk otomasi pabrik Mitsubishi Electric

CF26 CF37

Pembaca kode yang mendukung CC-Link IE Field Network Basic, SLMP, dan iQSS*1 kompatibel dengan pengontrol terprogram Mitsubishi Electric. Mengontrol, memantau, dan mengeluarkan hasil pembaca kode dapat dilakukan hanya dengan menghidupkan/mematikan data perangkat pengontrol terprogram.

*1.Untuk detailnya, silakan lihat “Katalog Solusi Sensor iQ Sensor Solution (L(NA)16029ENG)”.

Kompatibel dengan CC-Link IE Field Network Basic

Dilengkapi dengan antarmuka CC-Link IE Field Network Basic, pengaturan untuk menghubungkan dengan pengontrol yang dapat diprogram dapat dilakukan dengan mudah dan hasil pembacaan dapat diperoleh melalui jaringan.

Transfer data mewujudkan keterlacakan

Gambar hasil pengenalan pembaca kode dapat ditransfer ke GOT (HMI) dan sistem IT. Hasil pembacaan termasuk gambar kode yang gagal dibaca dan gambar yang dikenali ditransfer ke sistem IT untuk disimpan, sehingga memungkinkan analisis dan diagnosis.

Kompatibel dengan iQSS

Mendukung iQSS, pembaca kode secara otomatis terdeteksi pada alat rekayasa. Fungsi tautan alat memungkinkan pengurangan total biaya dalam pengembangan/penyiapan/pengoperasian/pemeliharaan.
