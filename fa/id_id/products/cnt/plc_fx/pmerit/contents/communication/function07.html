MELSEC-F Series Komunikasi peralatan periferal

  • 製品情報

Fitur

Dengan menggunakan perangkat komunikasi RS-422, USB, atau RS-232C, port koneksi untuk perangkat periferal dapat ditambahkan.
Perangkat untuk pemrograman sekuensial termasuk komputer pribadi dan unit tampilan (GOT) dapat dihubungkan ke port tambahan.

Contoh Konfigurasi Sistem

Perangkat komunikasi RS-422

  • *1Hanya dapat dihubungkan ke FX3UC-32MT-LT(-2).
  1. <Kabel koneksi>
  2. Gunakan kabel RS-422 yang telah ditentukan (tipe MINI DIN 8-pin pada sisi PLC) untuk setiap perangkat periferal.

Perangkat komunikasi USB

  • *1Hanya dapat dihubungkan ke FX3UC-32MT-LT(-2).

Perangkat komunikasi RS-232C

  • *1Hanya dapat dihubungkan ke adaptor tertentu.
  • *2Papan ekspansi fungsi hanya dapat dihubungkan ke FX3UC-32MT-LT(-2).
  1. <Kabel koneksi>
  2. Kabel koneksi utama untuk perangkat komunikasi RS-232C dan perangkat periferal tercantum di bawah ini.
    Untuk kombinasi lainnya, silakan lihat Panduan Pengguna Seri FX (Edisi Komunikasi Data).

Penggunaan perangkat periferal secara bersamaan

Hubungkan alat pemrograman seperti perangkat lunak PC ke salah satu perangkat periferal untuk menghindari program diubah oleh beberapa perangkat periferal.
