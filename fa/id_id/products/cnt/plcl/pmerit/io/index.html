Blok terminal klem pegas (tipe dorong): L6TE-18S

Blok terminal sekrup modul yang terpasang dapat diganti dengan blok terminal klem pegas tipe dorong.
Jenis blok terminal ini membantu mengurangi jumlah pemasangan kabel dan waktu perawatan.

■ Tipe push-in untuk mengurangi kabel

Lebih mudah untuk memasang kabel hanya dengan memasukkannya ke blok terminal.

■ Mudah untuk mengonfirmasi integritas sinyal

Termasuk terminal khusus untuk memasukkan probe uji, misalnya.
