Kumpulkan data produksi dengan mudah

Dengan memanfaatkan kartu memori SD yang terpasang atau koneksi langsung ke modul CPU, data pencatatan dapat dengan mudah direalisasikan hanya dengan mendaftarkan parameter. Data yang dicatat dapat disimpan dalam format CSV dan digunakan dalam sejumlah cara, seperti untuk digunakan pada perangkat lunak spreadsheet pihak ketiga atau sebagai data umpan waktu nyata untuk menganalisis berbagai proses manufaktur. Fitur waktu nyata GX LogViewer juga memungkinkan umpan langsung yang menunjukkan perubahan status perangkat, membantu meningkatkan keterlacakan, kelancaran pengaktifan, dan penelusuran kesalahan.

Pencatatan varians data kontrol

Data dikumpulkan selama setiap pemindaian atau dalam interval milidetik yang memungkinkan deteksi deviasi kontrol bahkan pada kecepatan yang sangat tinggi. Oleh karena itu, identifikasi kesalahan dapat dilakukan lebih cepat dan lebih rinci.

■ Data sampel generik dari PC atau perangkat eksternal pada interval 100 ms

■ Fungsi pencatatan data Seri L mampu mengambil sampel data pada interval yang jauh lebih tinggi untuk mendeteksi nilai yang berubah dengan cepat.

Fungsi pencatatan otomatis

Pencatatan data otomatis dapat dilakukan hanya dengan memasukkan kartu memori SD ke dalam CPU, yang dapat dilakukan karena kartu memori menyertakan berkas konfigurasi pencatatan. Instruksi pencatatan data dari jarak jauh juga dapat dilakukan hanya dengan mengirimkan berkas konfigurasi melalui email dan menyalinnya ke kartu memori SD.

Kirim file pencatatan secara otomatis ke server FTP

File pencatatan data yang disimpan di kartu memori SD dapat dikirim ke server FTP hanya dengan melakukan pengaturan sederhana menggunakan alat konfigurasi pencatatan. Karena server pencatatan dapat menangani banyak file, tugas pengelolaan dan pemeliharaan dapat dikurangi.

■ Fitur transfer file*2

  • *2.Menggunakan modul CPU dengan 5 digit pertama nomor seri "12112" atau yang lebih baru.

Fungsi pemicu pencatatan

Penyebab dan solusi kesalahan dapat dilakukan dengan cepat karena hanya data yang diperlukan terkait masalah yang diekstraksi, tanpa harus menghabiskan waktu untuk memfilter data diagnostik dalam jumlah besar.
