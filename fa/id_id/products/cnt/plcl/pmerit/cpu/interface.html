Koneksi USB dan Ethernet standar

Gunakan antarmuka USB 2.0 atau Ethernet untuk menghubungkan langsung di lokasi pemasangan. Antarmuka Ethernet mendukung koneksi langsung dengan kabel LAN silang atau lurus dan tidak memerlukan konfigurasi pengontrol yang dapat diprogram atau PC untuk beroperasi.

CC-Link IE Field Network Basic tidak memerlukan modul jaringan<sup>Fungsionalitas yang ditingkatkan!</sup>

CPU pengontrol yang dapat diprogram dengan port Ethernet tertanam dapat digunakan sebagai stasiun induk*1, sehingga menghilangkan kebutuhan akan modul jaringan tambahan. Jaringan dapat dikonfigurasi dengan jumlah modul minimum yang mengurangi biaya ruang dan perangkat keras.

  • *1.Hanya LnCPU(-P/-BT/-PBT) yang kompatibel.
  • *2.SLMP: Protokol Pesan Tanpa Batas

<span>Terhubung dengan mudah ke BACnet<sup class="vis">®</sup> dan MODBUS<sup class="vis">®</sup>/TCP<sup>Peningkatan fungsionalitas!</sup></span>

Ethernet mewujudkan koneksi berkecepatan tinggi, seperti komunikasi dengan perangkat eksternal.
Dengan menggunakan fungsi dukungan protokol yang telah ditetapkan sebelumnya, berbagai perangkat yang memerlukan dukungan protokol jaringan terbuka, seperti BACnet® dan MODBUS®/TCP didukung.

Stempel waktu jaringan

Sinkronkan sistem pada jaringan Ethernet menggunakan server SNTP*3. Sinkronisasi waktu dapat dilakukan untuk memungkinkan operasi simultan, kontrol kualitas, atau pelacakan kesalahan.

  • *3.SNTP: Simple Network Time Protocol

Transfer data perangkat tanpa program

Fungsi komunikasi PLC sederhana*4

Dengan menggunakan alat pemrograman, pengaturan parameter sederhana sudah cukup untuk mentransfer data perangkat seperti informasi produksi tanpa memerlukan pemrograman.
Fungsi ini memungkinkan untuk dengan mudah membangun komunikasi tidak hanya dengan Seri L, tetapi juga dengan pengontrol MELSEC Seri iQ-R, Seri iQ-F, Seri Q, dan Seri QnA/A.

  • *4.Modul CPU yang lima digit nomor seri pertamanya adalah "13042" atau yang lebih baru diperlukan.

Fitur khusus kartu memori SD

Gunakan kartu memori yang kompatibel dengan SD/SDHC untuk mencadangkan program dan parameter CPU dengan cepat dan mudah.
Cadangan tersebut kemudian dapat dengan mudah dikembalikan atau digunakan untuk memprogram CPU lain. Kartu memori juga dapat digunakan untuk menyimpan data yang diambil dengan fungsi pencatatan data*7.

  • *7.Untuk detail tentang fungsi pencatatan data, silakan lihat “Tren historis dan umpan langsung produksi”.

Simpan/muat program langsung ke Programmable Controller

Fungsi simpan/muat beberapa proyek*8

Parameter, file program, dll., dapat disimpan/dibaca ke kartu memori SD hanya dengan menggunakan unit tampilan internal, tanpa harus terhubung ke PC terpisah. Setelah disimpan di kartu memori SD, file dapat dikirim melalui email, misalnya, saat memerlukan pengeditan file di luar lokasi.

  • *8.Didukung oleh modul CPU yang lima digit nomor seri pertamanya adalah "14042" atau yang lebih baru.
