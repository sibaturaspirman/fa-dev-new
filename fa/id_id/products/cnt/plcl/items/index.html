Programmable Controllers
MELSEC-L Series

  • Katalog Produk

Seri MELSEC-L akan memberikan inovasi dan pengembangan pada lokasi produksi Anda.
Perspektif lokasinya memberikan kemudahan penggunaan yang luar biasa.

Seri MELSEC-L dirancang berdasarkan hasrat terhadap monozukuri (kreasi), teknologi tepercaya, dan keandalan. Konsep ini juga berlaku untuk pemasangan dan pengoperasian sistem.
Dengan berbagai fungsi I/O sebagai fitur standar, Seri MELSEC-L berfungsi untuk meningkatkan efisiensi kerja dengan mewujudkan kenyamanan, kemudahan penggunaan, dan kontrol yang beragam.

Daftar Produk

CPU module

Modul CPU untuk PLC.
Berbagai fungsi I/O disertakan sebagai standar.

Branch/extension module

Modul ini memungkinkan berbagai modul ditambahkan untuk memperluas sistem sesuai dengan ukuran fasilitas produksi Anda.

Power supply module

Modul ini digunakan untuk menyediakan daya listrik untuk CPU, input, output, dan modul lainnya pada unit dasar.

I/O module

Modul I/O digital mengintegrasikan sinyal bit ke sistem kontrol.

Analog module

Modul analog mencakup modul I/O analog yang menghubungkan sinyal tegangan dan arus ke sinyal kontrol, modul input suhu yang mengukur suhu, dan modul kontrol suhu yang mengendalikan suhu.

Simple motion/Positioning module

Modul gerakan sederhana mewujudkan kontrol gerakan berkecepatan tinggi dan berakurasi tinggi dengan pemrograman sederhana.
Modul pemosisian melakukan pemosisian seperti interpolasi heliks.

High-speed counter/Flexible high-speed I/O control module

Modul penghitung kecepatan tinggi mampu menangani sinyal pulsa kecepatan tinggi dari encoder.
Modul kontrol I/O kecepatan tinggi yang fleksibel mewujudkan respons I/O yang stabil karena pemrosesan perangkat keras kecepatan tinggi tanpa dibatasi oleh waktu pemindaian CPU. Selain itu, logika pengguna dapat diatur ke FPGA dengan alat konfigurasi intuitif khusus.

Network module

Modul jaringan mendukung berbagai jaringan seperti CC-Link IE dan CC-Link.

Option

Kami juga menyediakan rangkaian lengkap produk periferal seperti konektor untuk modul input/output dan hub industri.
