Programmable Controllers
MELSEC-Q Series

  • Katalog Produk

Dengan proses perintah dasar berkecepatan nano-order, Seri MELSEC-Q generasi berikutnya secara dramatis meningkatkan kinerja sistem dan mesin.

Seiring dengan terus berkembangnya peralatan dan fasilitas manufaktur setiap harinya, seri ini memungkinkan pemrosesan data dan kontrol mesin berkecepatan tinggi, berakurasi tinggi, dan bervolume besar.

CPU module

Modul CPU untuk PLC.
Jajaran CPU dapat disesuaikan dengan aplikasi apa pun.

Base unit

Untuk memasang perangkat seperti catu daya dan modul CPU.

Power supply module

Modul ini digunakan untuk menyediakan daya listrik untuk CPU, input, output, dan modul lain pada unit dasar.

I/O module

Modul I/O digital mengintegrasikan sinyal bit ke sistem kontrol.

Analog module

Modul analog mencakup modul I/O analog yang menghubungkan sinyal tegangan dan arus ke sistem kontrol, modul masukan suhu yang mengukur suhu, dan modul kontrol suhu yang mengendalikan suhu.

Simple motion/Positioning module

Modul gerakan sederhana mewujudkan kontrol gerakan berkecepatan tinggi dan berakurasi tinggi dengan pemrograman sederhana. Modul pemosisian melakukan pemosisian seperti interpolasi heliks.

High-speed counter/Pulse I/O module

Modul penghitung kecepatan tinggi mampu menangani sinyal pulsa kecepatan tinggi dari enkoder.
Modul masukan pulsa melakukan penghitungan kumulatif dengan memasukkan sinyal pulsa pengukuran seperti laju aliran.

Information/Network module

Modul informasi memungkinkan pertukaran informasi dengan basis data MES dan melakukan pencatatan data.
Modul jaringan mendukung berbagai jaringan seperti CC-Link IE dan CC-Link.

Energy measuring module

Modul pengukuran energi mengukur dan memantau berbagai informasi energi.
Modul pemantauan isolasi memantau kerusakan isolasi.

Option

Kami juga menyediakan jajaran produk periferal yang luas seperti konektor untuk modul input/output dan hub industri.
