Kompatibilitas

Gunakan modul Seri Q yang sudah ada

Modul Seri Q konvensional kompatibel dengan Seri QCPU model Universal. Oleh karena itu, saat memerlukan pemutakhiran, biaya perawatan sistem dari sistem yang sudah ada dapat ditekan seminimal mungkin dengan sedikit gangguan saat penggantian.

  • *1.Q00UJCPU adalah tipe all-in-one, dengan catu daya terintegrasi, unit dasar 5-slot, dan CPU.

Gunakan program Seri Q yang ada

Program QCPU konvensional dapat digunakan hanya dengan mengubah tipe PLC*2 di dalam alat pemrograman, yang memungkinkan peningkatan mudah ke Seri model Universal dengan sedikit rekayasa ulang yang diperlukan.

  • *2.Tergantung pada programnya, jumlah langkah dapat bervariasi saat jenis PLC diubah.

Cadangkan data penting secara otomatis

Program dan file parameter secara otomatis dicadangkan ke memori program (Flash ROM) yang tidak memerlukan cadangan baterai. Hal ini mencegah hilangnya data program dan parameter karena kegagalan dalam penggantian baterai. Selain itu, cadangan data penting seperti data perangkat dapat didaftarkan ke ROM standar untuk mencegah hilangnya data karena baterai habis jika terjadi pemadaman listrik yang direncanakan selama liburan berturut-turut. Data cadangan dipulihkan secara otomatis saat daya dipulihkan.

Memperpendek waktu pemulihan sistem saat mati

Fungsi penggantian modul CPU*3

Fungsi penggantian modul CPU memungkinkan pengguna membuat cadangan lengkap semua informasi CPU ke kartu memori. Jika terjadi kegagalan CPU atau kejadian bencana lainnya, data cadangan dapat digunakan untuk memprogram modul CPU baru dengan cepat.
Dengan menggunakan fungsi ini, sistem dapat segera dioperasikan dan waktu henti dapat diminimalkan.

  • *3.Tidak didukung oleh Q00UJ, Q00U and Q01U.

Nomor seri

Nomor seri dapat diperiksa dengan cepat tanpa harus melepaskannya dari unit dasar (Tidak perlu mengganggu pengoperasian).
Selain itu, nomor seri dapat diperiksa menggunakan fitur "daftar informasi produk" yang disertakan dalam GX Developer dan GX Works2.
