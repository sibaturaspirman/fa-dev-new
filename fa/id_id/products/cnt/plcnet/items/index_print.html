CC-Link IE TSN

Sistem Jaringan terintegrasi terbuka berbasis Ethernet yang memungkinkan komunikasi mulus antara sistem TI dan Area Produksi.

  • Fitur Produk

CC-Link IE Control Network

Jaringan pengontrol-ke-pengontrol berbasis Ethernet ini memungkinkan komunikasi berkecepatan tinggi dan transmisi volume tinggi, dengan komunikasi mulus yang tidak terputus oleh lapisan.

  • Fitur Produk

CC-Link IE Field Network

CC-Link IE menggabungkan kontrol I/O berkecepatan tinggi, kontrol terdistribusi generik, kontrol gerakan sinkron, dan kontrol keselamatan yang memungkinkan komunikasi keselamatan lintas beberapa perangkat keselamatan, semuanya pada jaringan yang sama.
Jaringan lapangan terbuka berbasis Ethernet ini dapat secara fleksibel mengonfigurasi sistem sesuai dengan lingkungan instalasi menggunakan kabel berpasangan terpilin yang sangat fleksibel.

  • Fitur Produk

CC-Link IE Field Network Basic

CC-Link IE Field Network Basic realizes easier network integration, as its cyclic communications stack is software-based, without requiring a dedicated ASIC helping to reduce implementation costs for device partners.
Transparent communications are achieved by utilizing SLMP that enables seamless connectivity within all levels of manufacturing.

  • Fitur Produk

CC-Link

World standard, first in Japan and in Asia.
This serial-based high-speed field network is capable of control and information handling.

CC-Link Safety

This safety field network constructs a safety system utilizing CC-Link high-speed communication to protect production sites from dangers.

CC-Link/LT

Jaringan pengurangan kabel internal dan eksternal yang menampilkan keterbukaan CC-Link untuk menghilangkan masalah di lokasi seperti pekerjaan pemasangan kabel yang terlalu rumit atau kesalahan pemasangan kabel.

Ethernet

Ethernet adalah jaringan tingkat tertinggi (tingkat manajemen) yang menghubungkan sistem atas dan lokasi produksi.

AnyWire

AnyWire adalah jaringan kontrol terdistribusi tingkat sensor yang dirancang untuk mengurangi biaya pemasangan dengan memanfaatkan kabel robot dan kabel serbaguna.

SSCNET

SSCNET is high-speed and high-performance servo system controller network utilizing optical fiber cable, which is extendable for a long distance.

MELSECNET/H

Modul jaringan yang menghubungkan pengontrol dalam setiap proses di lini produksi.

MODBUS

Modul antarmuka yang mendukung komunikasi MODBUS® dan MODBUS® /TCP.

EtherNet/IP

Interface module for EtherNet/IP™ network.

PROFIBUS

Modul antarmuka untuk jaringan PROFIBUS®.

DeviceNet

Modul antarmuka untuk jaringan DeviceNet®.

Serial communication

Modul komunikasi serial yang memungkinkan pengumpulan data PLC dengan menghubungkan ke perangkat yang mendukung RS-232/422/485.

FL-net(OPCN-2)

Modul antarmuka dapat dihubungkan ke jaringan FL-net.

AS-i

Modul antarmuka untuk jaringan AS-i.
