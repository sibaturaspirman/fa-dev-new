Modul Jarak Jauh Tipe Blok

Modul jarak jauh tipe Blok Jaringan Lapangan CC-Link IE

  • Stasiun perangkat jarak jauh atau stasiun perangkat cerdas Jaringan Lapangan CC-Link IE. Modul-modul ini berguna saat posisi pemasangan yang dekat dengan perangkat I/O diperlukan
  • Mendukung komunikasi tersinkronisasi CC-Link IE Field Network (Dengan melakukan sinkronisasi dengan stasiun induk*1*2, yang mendukung komunikasi tersinkronisasi, modul-modul ini melakukan operasi sinkron yang sangat akurat sebagai stasiun perangkat.)
  • Modul yang mendukung fungsi ekstensi meningkatkan jumlah titik I/O dengan menambahkan modul ekstensi
  • Modul*3 yang mendukung fungsi penghubungan cepat akan kembali dengan cepat saat disambungkan kembali dengan CC-Link IE Field Network setelah pemutusan sambungan
  • Modul*3 yang mendukung fungsi pengaturan parameter I/O otomatis dapat dioperasikan tanpa pengaturan parameter, sehingga mengurangi waktu penyalaan
  • *1.Stasiun induk MELSEC iQ-R Series dan modul gerak sederhana mendukung fitur ini.
  • *2.Saat menggunakan QD77GF16, fungsi ini tidak dapat digunakan tergantung pada kombinasi modul I/O dan paket perangkat lunak yang digunakan. Untuk perincian lebih lanjut, silakan rujuk ke manual modul jarak jauh tipe blok.
  • *3.Untuk modul yang berlaku, silakan rujuk ke daftar fungsi

Modul dengan  dikenali sebagai stasiun perangkat CC-Link IE TSN dengan mengubah sakelar di bagian depan modul. Silakan merujuk ke Jaringan CC-Link IE TSN.

Modul masukan utama

  • Waktu respons dapat diatur pada 0 ms, 0,2 ms, 0,5 ms, 1 ms, 1,5 ms, 5 ms, 10 ms, 20 ms, dan 70 ms

Modul ekstensi

Modul input/output

  • Meningkatkan jumlah titik I/O yang tersedia untuk modul I/O jarak jauh, I/O analog, dan penghitung kecepatan tinggi

  • Ketika modul output ekstensi terhubung ke modul input utama, modul output utama atau modul gabungan I/O, fungsi integrasi waktu ON dapat digunakan
