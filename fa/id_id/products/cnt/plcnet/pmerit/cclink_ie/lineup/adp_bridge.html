Modul jembatan CC-Link IE TSN - Jaringan Lapangan CC-Link IE

  • NZ2GN-GFBBARU
  • Memungkinkan akses dari CC-Link IE Field Network ke setiap stasiun dan perangkat di CC-Link IE TSN
  • Dapat digunakan sebagai stasiun induk atau lokal di CC-Link IE Field Network dan sebagai stasiun jarak jauh di CC-Link IE TSN
  • Memungkinkan perangkat CC-Link IE TSN ditambahkan ke peralatan yang ada di CC-Link IE Field Network *1
  • *1.Karena NZ2GN-GFB tidak berfungsi sebagai stasiun induk CC-Link IE TSN, diperlukan stasiun induk terpisah pada jaringan CC-Link IE TSN.
