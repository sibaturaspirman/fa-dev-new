CC-Link IE Field Network Basic

Seiring dengan tren terkini Industrial Internet of Things (IIoT)*1, koneksi jaringan perangkat dan peralatan untuk sistem skala kecil semakin menjadi hal yang umum. CC-Link IE Field Network Basic mewujudkan integrasi jaringan yang lebih mudah, karena tumpukan komunikasi sikliknya berbasis perangkat lunak, tanpa memerlukan ASIC khusus sehingga membantu mengurangi biaya implementasi bagi mitra perangkat. Mitsubishi Electric meluncurkan produk-produk kompatibel CC-Link IE Field Network Basic untuk semakin meningkatkan pemanfaatan jaringan di area produksi.

  • *1.Industrial Internet of Things

Komunikasi Tanpa Hambatan di Seluruh Pabrik

Dengan memanfaatkan teknologi Ethernet standar dan tumpukan protokol TCP/IP untuk komunikasi (seperti HTTP, FTP), SLMP mengalirkan data secara transparan antara tingkat sensor dan tingkat perusahaan di berbagai jaringan otomatisasi standar industri.
Komunikasi tanpa hambatan dapat dengan mudah diwujudkan dengan CC-Link IE Field Network Basic, yang selanjutnya meningkatkan kinerja perusahaan manufaktur.

Posisi dalam Jaringan CC-Link IE

Jaringan terbuka berbasis Ethernet, CC-Link IE, merupakan jaringan berkecepatan tinggi dan berkapasitas besar yang mengintegrasikan kontrol terdistribusi, I/O control, kontrol keamanan, dan kontrol gerak.
CC-Link IE Field Network Basic, sebagai bagian dari jaringan CC-Link IE, mewujudkan koneksi jaringan yang lebih mudah untuk perangkat Ethernet.
Komunikasi transparan tercapai dengan memanfaatkan SLMP yang memungkinkan konektivitas tanpa hambatan di semua tingkatan manufaktur.
