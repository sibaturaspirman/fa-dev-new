Modul jarak jauh tipe blok

Modul jarak jauh tipe blok dikenali sebagai stasiun perangkat pada CC-Link IE TSN. Modul ini terutama digunakan saat instalasi mengharuskannya berada dekat dengan I/O yang terhubung untuk menghemat kabel. Modul I/O digital merupakan indera dari sistem otomasi dan dapat dengan mudah dihubungkan ke sakelar, lampu indikator, sensor, dan perangkat lainnya. Modul analog dapat dihubungkan ke perangkat yang memproses tegangan dan sinyal arus yang bervariasi. Kombinasi modul jarak jauh tipe blok dengan fungsi keamanan dan modul CPU keamanan mewujudkan komunikasi jaringan keamanan.

  • Atur parameter dengan mudah hanya menggunakan sakelar perangkat keras*1
  • Deteksi tegangan catu daya rendah
  • Blok terminal penjepit pegas mengurangi kabel
  • Modul input sesuai dengan IEC 61131-2 Tipe3 (rentang operasi standar input digital), mendukung berbagai sensor*1
  • Beralih ke mode stasiun perangkat CC-Link IE Field Network*1
  • Modul jarak jauh tipe blok dengan fungsi keamanan memungkinkan kontrol keamanan
  • Waktu input/output dapat disinkronkan dengan siklus sinkronisasi antar-modular, memungkinkan kontrol yang sangat akurat sistem*1

Memulai sistem dengan mudah<sup>*1</sup>

Alamat IP untuk setiap modul dapat diatur dengan mudah menggunakan tombol-tombol di bagian depan modul. Fungsi-fungsi tambahan juga dapat diatur menggunakan tombol-tombol, tanpa memerlukan perangkat lunak rekayasa khusus.

Beralih ke mode stasiun perangkat CC-Link IE Field Network<sup>*1</sup>

Mengatur sakelar di bagian depan modul memungkinkan untuk digunakan sebagai stasiun perangkat CC-Link IE TSN atau CC-Link IE Field Network tanpa memerlukan modul terpisah.

  • *1.Modul jarak jauh tipe blok dengan fungsi keselamatan tidak didukung.

Modul input

  • Waktu respons dapat diatur pada 0 ms, 0,2 ms, 1 ms, 1,5 ms, 5 ms, 10 ms, 20 ms, dan 70 ms (pengaturan awal adalah 1 ms)
  • Tegangan dan arus ON/OFF input mematuhi IEC 61131-2 (rentang operasi standar input digital) Tipe 3, yang mendukung berbagai sensor
  • Fungsi dapat dengan mudah diatur dari bagian depan modul menggunakan sakelar, tanpa memerlukan perangkat lunak rekayasa khusus

Modul FPGA<sup>BARU</sup>

  • Waktu respons I/O yang sangat akurat dan sangat cepat dalam hitungan mikrodetik dengan field programmable gate array (FPGA)
  • Pengoperasian nilai I/O diproses dalam hitungan nanodetik di FPGA. Rangkaian logika dapat diprogram secara fleksibel menggunakan perangkat lunak Intel® Quartus® Prime Design*3
  • Berbagai jenis komunikasi tersedia untuk komunikasi Ethernet dengan sistem tingkat tinggi, yang mana salah satu yang sesuai dapat dipilih menurut aplikasi dan kondisi. Modul ini juga dapat beroperasi secara mandiri selama start-up atau untuk fasilitas insidental
  • Rangkaian logika kompleks yang dirancang pelanggan dapat ditulis ke FPGA. Modul ini ideal untuk mengganti papan FPGA dan papan mikrokomputer yang sudah ada yang dikembangkan di dalam negeri
  • Data pencatatan dapat dibuat sebagai berkas CSV dan ditransfer ke server FTP menggunakan FTP (klien). FTP dapat digunakan secara bersamaan dengan CC-Link IE TSN
  • Modul ini mendukung komunikasi CPU yang sederhana, yang memungkinkan pertukaran data dengan modul CPU
  • Tabel di bawah ini menunjukkan spesifikasi FPGA tertanam
