CPU module

Catu daya, CPU, dan unit utama I/O all-in-one. Fitur bawaan meliputi penghitung kecepatan tinggi, keluaran pemosisian, Ethernet, dan slot kartu SD.

I/O module

Modul I/O dengan catu daya terintegrasi dan tanpa catu daya tersedia dalam berbagai ukuran dan konfigurasi I/O.

Analog Control module

Opsi untuk input dan output analog, input sensor suhu, dan kontrol suhu. Modul fungsi cerdas analog seri FX3 juga dapat dihubungkan.

High-Speed Counting module

Modul penghitung kecepatan tinggi. Modul penghitung kecepatan tinggi seri FX3 juga dapat dihubungkan.

Motion/Positioning Control module

Modul gerakan sederhana yang mendukung SSCNET III/H. Modul keluaran pulsa seri FX3 juga dapat dihubungkan.

Network/Communication module

Modul stasiun perangkat cerdas CC-Link IE Field Network. Modul komunikasi RS-232, RS-485, RS-422. Modul induk CC-Link seri FX3 juga dapat dihubungkan.

Safety extension module

Modul ekstensi keamanan. Buat sistem dengan mudah hanya dengan menghubungkan modul ekstensi keamanan ke modul CPU FX5UJ/FX5U/FX5UC.

Option

Modul konversi bus, catu daya, jajaran baterai.
