Konsep

Buat sistem dengan mudah hanya dengan menghubungkan modul ekstensi pengaman.

Sistem kontrol pengaman dapat dipasang dengan mudah hanya dengan menghubungkan modul utama pengaman (FX5-SF-MU4T5) ke modul CPU FX5U/FX5UC/FX5UJ. Sistem tunggal ini kemudian dapat digunakan untuk melakukan kontrol serbaguna dan kontrol pengaman. Oleh karena itu, tidak diperlukan kabel seperti yang diperlukan untuk memantau status pengaman (seperti yang diperlukan dengan pengontrol pengaman) atau kabel yang diperlukan di antara relai saat membangun sistem dengan relai pengaman.
Lebih jauh, jumlah input pengaman dapat diperluas dengan menghubungkan modul ekspansi input pengaman (FX5-SF-8DI4).

Putar sakelar putar untuk memilih program bawaan.

Setiap modul ekstensi keselamatan memiliki sembilan jenis program bawaan. Untuk membangun sistem kontrol keselamatan, cukup gunakan sakelar putar di bagian depan modul untuk memilih program bawaan yang akan dijalankan.
Ini menghilangkan kebutuhan akan program berurutan yang dirancang untuk kontrol keselamatan.
