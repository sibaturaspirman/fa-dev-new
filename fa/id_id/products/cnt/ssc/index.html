Sistem Jaringan

CC-Link IE TSN

Sistem Jaringan terintegrasi terbuka berbasis Ethernet yang memungkinkan komunikasi mulus antara sistem TI dan Area Produksi.
CC-Link IE Field Network

CC-Link IE menggabungkan kendali I/O berkecepatan tinggi, kendali terdistribusi generik, kendali gerakan serempak, dan kendali keselamatan yang memungkinkan komunikasi keselamatan di beberapa perangkat keselamatan, semuanya dalam satu jaringan. Jaringan lapangan terbuka berbasis Ethernet ini dapat mengkonfigurasi sistem secara fleksibel sesuai dengan lingkungan instalasi menggunakan kabel twisted-pair yang sangat fleksibel.
CC-Link IE Field Network Basic

CC-Link IE Field Network Basic mewujudkan integrasi jaringan yang lebih mudah, karena tumpukan komunikasi siklisnya berbasis perangkat lunak, tanpa memerlukan ASIC khusus yang membantu mengurangi biaya implementasi bagi mitra perangkat. Komunikasi transparan dicapai dengan memanfaatkan SLMP yang memungkinkan konektivitas mulus di semua tingkat manufaktur.
SSCNET III/H

SSCNET is high-speed, high-performance, and high-reliability servo system controller network utilizing optical fiber cable, which is extendable for a long distance.
