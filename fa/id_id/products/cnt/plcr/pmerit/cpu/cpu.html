Modul CPU pengontrol yang dapat diprogram

Modul CPU merupakan inti dari sistem kontrol yang menjalankan berbagai tugas kontrol. Berbagai CPU dengan kapasitas program dari 10K hingga 1200K langkah tersedia berdasarkan kebutuhan sistem. Dilengkapi dengan berbagai fungsi internal seperti server web dan basis data, yang berkontribusi pada peningkatan produktivitas.

Jaringan tertanam

Modul CPU pengontrol terprogram tertanam CC-Link IE.
Dua port Ethernet di sisi jaringan dapat digunakan sebagai gateway. Port Ethernet di sisi CPU digunakan sebagai port komunikasi Ethernet.

Antarmuka

Fitur<sup>Pemrograman mudah dengan penyimpanan data fleksibel dan berkapasitas besar</sup>

  • Maks. 9882 ribu kata
  • Pemrograman yang mudah
  • Pemanfaatan data
  • Modul CPU pengontrol terprogram Seri MELSEC iQ-R dirancang untuk memungkinkan kaset SRAM yang diperluas dipasang langsung ke dalam modul CPU. Opsi ini memungkinkan peningkatan memori perangkat internal hingga 9882K kata, memperluas memori perangkat/label lebih jauh lagi
  • Manajemen data internal pengontrol terprogram cukup fleksibel, membuat pemrograman menjadi lebih mudah dengan memungkinkan berbagai alokasi area data diubah dalam memori CPU dan kaset SRAM
  • Penggunaan kartu memori SD memperluas memori pencatatan data, yang memungkinkan analisis data oleh komputer

Mendeteksi sinyal kecepatan tinggi secara akurat

  • Interupsi konstan
  • Waktu interval minimum 50 µs
  • Kontrol kecepatan tinggi
  • Menjalankan program interupsi pada siklus konstan (waktu interval minimum 50 µs) yang berbeda dari waktu pemindaian
  • Semua sinyal terdeteksi karena pemrosesan input dapat dilakukan lebih cepat daripada waktu pemindaian. Ini ideal untuk jalur produksi tempat produk melaju dengan kecepatan tinggi

Riwayat kejadian yang terekam dapat diperiksa saat terjadi kesalahan

  • Tampilan dalam rangkaian waktu
  • Identifikasi penyebab kesalahan
  • Pemecahan Masalah
  • Kejadian historis operasi dan sistem direkam secara otomatis dalam modul CPU pengontrol terprogram
  • Perubahan aktual pada program dan kejadian sistem dengan kode kejadian yang sesuai dapat ditampilkan secara berurutan, memungkinkan identifikasi penyebab kesalahan dan pemecahan masalah dengan cepat

Kumpulkan data manufaktur dengan mudah

  • Simpan di memori internal
  • Ketertelusuran
  • Debug
  • Dengan memanfaatkan kartu memori SD yang terpasang, memori internal CPU, atau koneksi langsung ke modul CPU, data pencatatan (perangkat/label) dapat dikumpulkan dengan mudah hanya dengan mendaftarkan parameter. Selain itu, pemantauan melalui USB atau Ethernet secara real-time juga dimungkinkan
  • Dengan fungsi monitor real-time dari GX LogViewer, perubahan kecil pada perangkat target dapat diidentifikasi dengan mudah, yang berguna untuk keterlacakan pabrik, pengaktifan peralatan, dan debugging jika terjadi kesalahan

Diagnostik/pemantauan mudah melalui peramban web

  • Diagnostik CPU
  • Pemantau blok perangkat
  • Halaman web yang disesuaikan
  • Fungsi server web yang terpasang di CPU memungkinkan akses mudah ke diagnostik CPU dan pemantau batch perangkat melalui peramban web PC atau tablet tanpa perlu membuat grafik
  • Diagnostik awal dapat dilakukan dengan mudah saat terjadi masalah, tanpa perlu menghubungkan perangkat lunak rekayasa GX Works3
  • Alat desain halaman web pengguna memungkinkan pelanggan membuat grafik mereka sendiri dengan mudah dengan mengatur berbagai komponen grafik secara bebas disediakan*2
  • *2.Untuk perincian tentang cara memperoleh perangkat lunak, silakan hubungi kantor penjualan atau perwakilan Mitsubishi Electric setempat.

Koordinasi data yang mudah dengan pengontrol terprogram pihak ketiga hanya dengan mendaftarkan parameter

  • Hanya pendaftaran parameter
  • Koordinasi dengan pengontrol pihak ketiga
  • Pengumpulan data yang mudah
  • Modul CPU pengontrol terprogram memungkinkan pertukaran data perangkat dengan pendaftaran parameter dengan pengontrol terprogram Mitsubishi Electric dan pengontrol terprogram pihak ketiga*3 (fungsi komunikasi CPU sederhana)*4
  • Pengumpulan data lebih mudah tanpa mengubah program pengontrol terprogram yang ada
  • *3.Jika R□ENCPU (bagian jaringan) atau RJ71EN71 digunakan, maka dimungkinkan untuk terhubung ke pengontrol terprogram pihak ketiga.
  • *5.Hanya didukung oleh port Ethernet tertanam.

Manajemen data yang mudah dengan memanfaatkan database internal (DB)

  • DB Internal
  • Unicode®
  • Berbagai variasi produk
  • Data resep dan data produksi, yang sebelumnya dikelola menggunakan server basis data, kini dapat dikelola melalui basis data dalam pengontrol yang dapat diprogram
  • Fitur ini memungkinkan pemilihan perintah basis data yang dapat menambah/memperbarui/mencari/menghapus rekaman untuk digunakan untuk fungsi resep sederhana
  • Mengakses data basis data internal CPU dari komputer dengan Access® atau Excel® didukung
  • Digunakan untuk mengubah data resep dan manajemen produksi dalam industri makanan dan minuman di mana beberapa variasi produk diproduksi menggunakan proses mesin yang sama

Modul tanpa baterai mengurangi biaya perawatan

  • Modul CPU pengontrol terprogram untuk sistem skala menengah (R00CPU/R01CPU/R02CPU) mencakup memori nonvolatil tertanam, sehingga menyimpan data kait perangkat dan label tanpa memerlukan baterai cadangan saat daya mati
  • Modul CPU pengontrol terprogram lainnya juga dapat menyimpan data kait tanpa baterai dengan memasang kaset opsi tanpa baterai
  • Penggantian baterai berkala tidak lagi diperlukan, sehingga mengurangi biaya perawatan
