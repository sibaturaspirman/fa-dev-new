CC-Link IE TSN - Modul Jembatan CC-Link
FA3-BR1TC

Modul jembatan ini digunakan untuk menghubungkan CC-Link ke CC-Link IE TSN dengan lancar.
Modul jembatan beroperasi sebagai stasiun jarak jauh pada CC-Link IE TSN, dan sebagai stasiun induk pada CC-Link.

  • Kontrol perangkat CC-Link dari perangkat CC-Link IE TSN (Kontrol perangkat CC-Link dari perangkat CC-Link IE TSN)
  • Penggunaan perangkat dan kabel yang ada membantu memangkas biaya dan mengurangi waktu konstruksi
  • Konfigurasi perangkat CC-Link yang ada dapat dibaca sebagai parameter dan tercermin pada pengaturan parameter otomatis CC-Link IE TSN. (Fungsi deteksi otomatis sistem CC-Link didukung.)

Modul antarmuka FL-net (OPCN-2)
ER-1FL2-T

FL-net adalah jaringan otomasi pabrik terbuka yang distandarkan oleh Japan FA Open Systems Promotion Group. Jaringan ini menghubungkan beberapa perangkat seperti pengontrol yang dapat diprogram, pengontrol numerik terkomputerisasi (CNC), dan komputer pribadi yang diproduksi oleh masing-masing produsen, untuk mencapai kontrol dan pemantauan.

  • Karena FL-net tidak memerlukan stasiun induk, setiap node dapat dihubungkan/diputuskan ke/dari jaringan kapan saja tanpa memengaruhi komunikasi dengan node lain. Hingga 254 perangkat*1 dapat dihubungkan
  • Fungsi yang didukung: Fungsi memori umum, yang memungkinkan semua node untuk terus berbagi data yang sama melalui transmisi siklik. Fungsi komunikasi pesan, yang memungkinkan node untuk bertukar data yang diperlukan hanya saat diperlukan
