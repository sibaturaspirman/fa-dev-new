Perangkat Lunak Rekayasa PLC Terpadu
Evolusi utama perangkat lunak rekayasa PLC

Sesuai dengan Standar Internasional IEC 61131-3

Kini perangkat lunak rekayasa yang mudah digunakan bukanlah hal yang mengejutkan.
Selain kegunaannya yang canggih, perangkat lunak rekayasa GX Works2 menerapkan konsep arus utama global "pengelompokan" dan "penataan" untuk peningkatan mendasar efisiensi pemrograman.
Gaya rekayasa standar dunia dimulai dengan GX Works2.

Memfasilitasi manajemen proyek

1Mencadangkan dan memulihkan proyek dengan mudah

Dengan mendaftarkan riwayat revisi proyek, proyek dapat dengan mudah dipulihkan ke kondisi semula.
Proyek dengan riwayat yang terdaftar akan dibandingkan.

Tidak perlu menyimpan proyek dengan nama berbeda untuk pencadangan.

2Tampilan judul program memandu Anda

Selain nama program, judul program juga ditampilkan, yang memungkinkan konten program dipahami sekilas.

3Tampilan pohon menawarkan alur pemrosesan yang mudah dipahami

Pernyataan yang ditambahkan ke proses program ditampilkan pada tampilan pohon untuk memudahkan akses ke sana.
Alur pemrosesan dan struktur program mudah dipahami dan beralih ke setiap proses dengan cepat.

4Menangani beberapa bagian program dengan Seri FX

Program PLC dapat dibuat dengan beberapa bagian program sehingga konfigurasi program dapat dilihat dan bagian-bagian tersebut dapat digunakan dengan mudah dalam proyek lain.

5Manfaatkan sepenuhnya layar yang lebar dan mudah dibaca

Jendela docking disembunyikan untuk menggunakan layar secara efisien.

6Pengaturan tujuan koneksi yang mudah

Pengaturan untuk perangkat yang sering terhubung dapat disimpan dan digunakan kembali kapan pun diperlukan.
Hal ini menghilangkan kebutuhan untuk menyalin dan memodifikasi proyek untuk target koneksi yang berbeda.

7Menyesuaikan pengaturan tombol keyboard

Pengguna dapat menyesuaikan pintasan keyboard.
Pengaturan yang disesuaikan juga dapat disimpan dan diekspor sebagai file.

Tombol pintas dapat ditetapkan ke menu yang tidak memiliki pintasan yang ditetapkan secara default.

8Informasi bantuan memandu metode operasi Anda

Menampilkan informasi Bantuan dengan satu penekanan tombol memudahkan konfirmasi operasi.

Layar bantuan yang sering digunakan ditandai.
