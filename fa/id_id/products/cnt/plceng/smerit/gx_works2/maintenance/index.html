Perangkat Lunak Rekayasa PLC Terpadu
Evolusi utama perangkat lunak rekayasa PLC

Sesuai dengan Standar Internasional IEC 61131-3

Kini perangkat lunak rekayasa yang mudah digunakan bukanlah hal yang mengejutkan.
Selain kegunaannya yang canggih, perangkat lunak rekayasa GX Works2 menerapkan konsep arus utama global "pengelompokan" dan "penataan" untuk peningkatan mendasar efisiensi pemrograman.
Gaya rekayasa standar dunia dimulai dengan GX Works2.

Menyediakan perawatan yang sesuai dengan kebutuhan Anda

1Fungsi verifikasi yang ditingkatkan

Verifikasi data proyek yang terbuka terhadap data proyek yang tersimpan untuk menampilkan hasil dalam format yang mudah dilihat.
Parameter dan program dalam PLC yang terhubung ke komputer pribadi juga diverifikasi terhadap data proyek yang terbuka.

Hasil verifikasi disimpan ke file CSV untuk memudahkan revisi dokumen desain.

2Cegah kesalahan edit dengan mode Baca dan Monitor

Operasi yang salah dalam pemantauan dan pencarian dihilangkan dengan mendukung mode Baca dan Monitor yang mirip dengan GX Developer.

Mode tulis/monitor (mode tulis)

Layar Enter Symbol terbuka dengan menekan tombol Enter.

Mode baca/mode monitor

Menemukan layar terbuka dengan menekan tombol Enter.

Operasi tombol yang sama seperti GX Developer digunakan untuk mengganti mode.

3Pemantauan khusus untuk modul fungsi cerdas

Saat menonton program ladder, sinyal memori buffer/XY dari modul fungsi cerdas dipantau di jendela docking. Karena nama setiap alamat memori buffer ditampilkan, maka tidak perlu merujuk ke manual untuk melihat untuk apa memori buffer digunakan.

4Fungsi monitor Sistem Visible dan diagnostik PLC

Status operasi seluruh sistem pengontrol terprogram ditampilkan dengan jelas.
Diagnosis dan informasi terperinci setiap modul ditampilkan pada monitor untuk seluruh sistem yang memungkinkan titik masalah dikonfirmasi dengan cepat.

[Fungsi pengumpulan riwayat kesalahan modul]

Riwayat kesalahan PLC dan modul fungsi cerdas dilihat dalam rangkaian waktu.

[Diagnostik PLC]

Riwayat kesalahan PLC segera diperiksa untuk segera menanggapi kegagalan. Operasi jarak jauh juga dilakukan pada CPU pengontrol yang dapat diprogram untuk menyetel ulang atau memformat memorinya.

[Informasi terperinci modul]

Menampilkan status modul, detail kesalahan, dan solusi untuk kesalahan tersebut. Respons segera diberikan terhadap kegagalan modul.

[Diagnostik jaringan]

Menampilkan status seluruh jaringan secara visual sehingga masalah jaringan dan kesalahan modul dapat segera ditemukan.
Selain itu, pemantauan sistem PLC di stasiun lain dimulai melalui jaringan.

Sistem didiagnosis pada layar grafis yang memberikan kesan seolah-olah Anda sedang mengamati sistem dan peralatan sebenarnya.

5Fungsi cetak yang kaya

Item yang akan dicetak ditentukan secara rinci. Selain itu, beberapa program dicetak dalam satu operasi.

6Simpan dan edit label dan parameter dengan Microsoft® Excel®

Berbagai data program diekspor sebagai berkas format CSV.

  • Mengekspor data program sebagai berkas format CSV memiliki keuntungan berikut:
  • Data dikonfirmasi bahkan pada komputer pribadi yang tidak memiliki GX Works2
  • Data disimpan di komputer pribadi
  • Data dikirim melalui pos ke lokasi terpencil
  • Penggunaan data sekunder, seperti dokumentasi dan grafik, dapat dilakukan menggunakan Microsoft® Excel®
  • Berkolaborasi dengan perangkat lunak lain dengan menangani data dalam format CSV

Contoh pengaturan penugasan I/O dalam file CSV
