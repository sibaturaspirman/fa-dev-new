Evolusi utama perangkat lunak rekayasa PLC

Sesuai dengan Standar Internasional IEC 61131-3

Kini perangkat lunak rekayasa yang mudah digunakan bukanlah hal yang mengejutkan.
Selain kegunaannya yang canggih, perangkat lunak rekayasa GX Works2 menerapkan konsep arus utama global "pengelompokan" dan "penataan" untuk peningkatan mendasar efisiensi pemrograman.
Gaya rekayasa standar dunia dimulai dengan GX Works2.

Mempromosikan standarisasi program

1Memudahkan penggunaan FB

Apa itu Function Block (FB)?

Function Block (FB) adalah blok ladder yang sering digunakan dalam program sekuensial dan dikelompokkan sebagai bagian untuk digunakan kembali dalam program.
FB meningkatkan efisiensi pengembangan program dan mengurangi kesalahan pemrograman untuk memastikan kualitas program yang lebih tinggi.

Membuat komponen

Contoh) Program proses penghitungan ini menyalakan sinyal keluaran (Y12) setelah sinyal masukan (X1) menyala sebanyak 12 kali.

Keuntungan menggunakan FB

Keuntungan1 : Pemrograman lebih mudah

Program sekuens dibuat hanya dengan menyeret dan meletakkan FB.
Hal ini secara signifikan mengurangi proses pengembangan program.

Keuntungan2 : Keterbacaan lebih baik

Penggunaan FB dalam program sekuensial meningkatkan keterbacaannya karena program tersebut hanya terdiri dari "kotak" (FB), masukan, dan keluaran.

Keunggulan3 : Dapat digunakan kembali

Dengan mengelompokkan komponen program yang sering digunakan sebagai bagian, komponen tersebut dapat digunakan kembali sebanyak yang diperlukan.
Operasi seperti menyalin program yang sudah ada dan memodifikasi perangkat tidak lagi diperlukan.

Keunggulan4 : Kualitas lebih tinggi

Dengan mengelompokkan komponen program yang sering digunakan sebagai bagian (FB) dan menggunakannya kembali, kualitas program akan seragam dan tidak bergantung pada tingkat keterampilan pengembang.

Keuntungan5 : Pencegahan pencurian

Dengan mengelompokkan komponen program urutan penting yang melibatkan keahlian teknologi sebagai satu bagian (FB) dan melindunginya dengan kata sandi, kebocoran informasi dapat dicegah.

2Perpustakaan FB yang berguna yang disediakan oleh vendor

Apa itu pustaka FB?

Pustaka FB adalah kumpulan komponen FB yang digunakan dalam proyek sederhana GX Works2.
Dengan menggunakan FB ini, pengaturan dan pengoperasian modul MELSEC-Q/L serta produk mitra dikonfigurasi.

Selain FB yang dibuat khusus, pustaka FB yang bermanfaat yang disediakan oleh mitra kami tersedia. FB juga ditawarkan untuk produk mitra iQSS.
Perpustakaan MELSOFT memiliki lebih dari 1500 FB dari empat belas perusahaan, dan dijadwalkan untuk terus berkembang.

Jika cara menggunakan FB tidak yakin, klik kanan pada Daftar Proyek untuk menampilkan informasi bantuan.
