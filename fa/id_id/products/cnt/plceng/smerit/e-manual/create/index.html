Fitur Perangkat Lunak Pengontrol yang Dapat Diprogram MELSEC Fitur e-Manual Create untuk konversi dan pembuatan manual

e-Manual adalah buku elektronik yang memungkinkan pengguna produk Mitsubishi Electric FA untuk memperoleh informasi yang diperlukan dengan cepat seperti jenis produk dan penggunaan.

Fungsi pencarian yang sangat baik dan keterkaitan dengan perangkat lunak teknik memudahkan pengaktifan dan pemeliharaan produk FA.
Manual pemeliharaan pengguna dapat dengan mudah diubah menjadi file e-Manual dan ditautkan ke manual produk Mitsubishi Electric FA, sehingga memungkinkan pemecahan masalah yang cepat.

Mendukung pemecahan masalah pengguna pada peralatan manufaktur

Informasi pemecahan masalah khusus produk FA dapat dicari dengan cepat dan dirujuk secara langsung. Informasi yang diperlukan dapat dikumpulkan dari beberapa manual produk FA, sehingga mengurangi waktu yang dihabiskan untuk membuat manual. Informasi dapat dibagikan dan ditautkan lebih dari sebelumnya.

Informasi dan pengetahuan terbaru dapat dibagikan dengan anggota tim lainnya menggunakan fungsi berbagi. Bookmark (favorit) dan catatan juga dapat dibagikan.

[Fungsi pembuatan manual] Memfasilitasi pembuatan e-Manual dari file Word® atau file CHM untuk semua pengguna

Hanya tiga langkah yang diperlukan untuk mengonversi dokumen yang ada ke file e-Manual di e-Manual Create

e-Manual Create mengonversi file Microsoft® Word® atau file CHM (Microsoft® Compiled HTML Help) ke dalam file e-Manual. Setelah dokumen pengguna dikonversi ke file e-Manual, dokumen tersebut dapat dilihat di e-Manual Viewer. Dokumen tersebut tersedia sebagai buku elektronik.

Membuat e-Manual dengan mudah dari file Word® atau file CHM oleh semua pengguna menggunakan sampel

Templat dan sampel e-Manual

Dengan menerapkan lembar gaya e-Manual dan menggunakan data sampel (file Word®, file CHM), semua pengguna e-Manual Create dapat membuat file e-Manual.
