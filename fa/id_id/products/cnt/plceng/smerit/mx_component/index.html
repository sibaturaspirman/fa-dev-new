PLC engineering software (MELSEC) MELSOFT MX Component

  • Katalog Produk

Komponen MX adalah pustaka kontrol Active X®/kontrol .NET yang memungkinkan komunikasi dari PC ke pengontrol yang dapat diprogram dan pengontrol gerakan tanpa memandang protokol komunikasi.
Program rumit untuk komunikasi serial dan Ethernet dapat dikembangkan dengan langkah-langkah sederhana.

Atur kondisi komunikasi dengan mudah menggunakan Wizard

Utilitas konfigurasi komunikasi bergaya Wizard memudahkan akses ke CPU pengontrol terprogram.
Utilitas konfigurasi komunikasi menyimpan nomor stasiun logika CPU pengontrol terprogram yang telah ditetapkan, sehingga memudahkan akses ke CPU pengontrol terprogram hanya dengan mengatur nomor stasiun.

Pengumpulan data oleh VBA

Aplikasi tampilan grafik waktu nyata dapat dibuat menggunakan pemrograman VBA di Excel® dan Access®.
Data perangkat pengontrol terprogram yang dicatat dapat dikumpulkan dan disimpan secara waktu nyata.

Kurangi jam kerja dengan mengembangkan program berlabel

Perangkat dapat diatur sesuai dengan label yang ditetapkan.
Label memungkinkan konfigurasi program yang intuitif dalam MX Sheet atau langsung dalam program itu sendiri.
Oleh karena itu, jika terjadi perubahan pada perangkat, tidak perlu lagi mengubah program atau berkas MX Sheet.
