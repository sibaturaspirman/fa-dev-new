Lingkungan rekayasa PLC telah mengalami inovasi dan pertumbuhan yang luar biasa.
Kita sekarang memasuki era Engineering software MELSEC.

Berbagai produk perangkat lunak MELSEC menyediakan solusi untuk pengurangan TCO dalam lingkungan rekayasa, menggunakan metode seperti meningkatkan efisiensi desain, memperpendek waktu debugging, mengurangi waktu henti, dan penyimpanan data.

Beberapa produk bersifat khusus regional; untuk ketersediaan penjualan terkini, silakan periksa halaman web lokal Anda.

FieldDeviceConfigurator

Perangkat lunak pengelolaan/pengaturan Field Device yang merupakan produk MITSUBISHI ELECTRIC dan sesuai dengan spesifikasi terbuka FDT/DTM. Dan dapat digunakan sebagai aplikasi kerangka FDT untuk mengatur parameter Field Device, atau untuk memelihara/menyesuaikan perangkat.
