Fitur

Grafik data pengukuran di browser web

Memvisualisasikan data Anda

Kemampuan server web memungkinkan Anda memeriksa penggunaan energi secara real-time dari PC di jaringan yang sama.

Kompatibel dengan ponsel pintar dan tablet

Grafik dapat dilihat dari perangkat seluler seperti ponsel pintar dan tablet serta PC.

Tidak perlu membuat grafik

Berbagai fungsi grafik bawaan memungkinkan manajemen penghematan energi yang sesuai dengan kebutuhan Anda.

Pemanfaatan data dan pemberitahuan jarak jauh dengan transfer data otomatis dan fungsi pemberitahuan email

Pemanfaatan data yang mudah

Data pengukuran dapat ditransfer secara otomatis ke server FTP. Hal ini memungkinkan Anda untuk memanfaatkan data dalam sistem host dan mencadangkan data pengukuran.

Percepatan siklus PDCA

EcoWebServerIII mengirimkan email untuk memberi tahu Anda saat target terlampaui atau status pengoperasian peralatan berubah. Hal ini memungkinkan deteksi dini kelainan dari jarak jauh.

Pemantauan permintaan terpusat dan manajemen data energi

Operasi dan manajemen

Pemantauan permintaan daya dan konsumsi energi lokal mengidentifikasi penyebab terjadinya lonjakan. Ini membantu menghemat energi dan menekan permintaan secara efisien.

Kontrol otomatis

EcoWebServerIII mengeluarkan peringatan saat target permintaan kemungkinan akan terlampaui. Ia juga secara otomatis menghentikan beban AC, lampu, dan perangkat lain untuk menekan permintaan.

Tautan sistem

Penekanan permintaan dapat dicapai dengan menghubungkan EcoWebServerIII dengan pengontrol AC Mitsubishi Electric AE-200J dan perangkat lainnya.

Perluasan sistem dan aplikasi dengan koneksi ke PLC dan terminal operasi grafis (GOT)

Improved productivity

Mengumpulkan informasi produksi yang dimiliki oleh PLC memungkinkan Anda mengelola intensitas bersama informasi energi dan melaksanakan aktivitas perbaikan khusus peralatan.

Meningkatkan kesadaran akan konservasi energi di lokasi

Data yang dikumpulkan dapat ditampilkan di GOT. Hal ini memungkinkan pemantauan di lokasi secara langsung, termasuk intensitas dan permintaan.
