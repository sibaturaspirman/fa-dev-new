Energy Measuring Units
(Model standar)

EMU4-BD1A-MB

  1. Dilengkapi dengan fungsi pengukuran energi dasar
  2. Komunikasi MODBUS RTU

Fungsi model standar plus fungsi tambahan untuk pengukuran tiga fase 4-kawat dan penghitungan pulsa

Energy Measuring Units
(Model kinerja tinggi)

EMU4-HD1A-MB

  1. Dilengkapi dengan fungsi pengukuran energi dasar
  2. Komunikasi MODBUS RTU
  3. Input tegangan langsung 440 V, 4-kawat, tiga fase
  4. Pengukuran nilai konversi energi listrik, energi listrik periodik, dan lainnya
  5. Pengukuran harmonik dan pemantauan pita

Pengukuran arus searah dari sisi sekunder (1 A atau 5 A) transformator arus serbaguna

Energy Measuring Units
(Model serbaguna)

EMU4-FD1-MB

  1. Konektivitas ke transformator arus serbaguna
  2. Komunikasi MODBUS RTU
  3. Komunikasi MODBUS TCP dengan unit opsional
  4. Input tegangan langsung 440 V, 3-kawat, tiga fase
  5. Tampilan arus harmonik, tegangan, arus fase nyata, energi listrik periodik, dan nilai konversi CO₂
  6. Fungsi input/output pulsa dan kontak

Optional Unit

Pengumpulan/pencatatan data pengukuran otomatis

Unit Pencatatan

EMU4-LM

Data pencatatan dari nilai terukur berbagai item pengukuran (energi listrik, tegangan, arus, dll.) dapat disimpan ke kartu SD sebagai file CSV

Berbagai macam koneksi transmisi/komunikasi

Unit komunikasi CC-Link

EMU4-CM-C

Ekstensi mudah ke sistem visualisasi EcoWebServer III atau sistem PLC melalui unit komunikasi CC-Link

Berbagai macam koneksi transmisi/komunikasi

Unit komunikasi dasar jaringan lapangan CC-Link IE

EMU4-CM-CIFB

Komunikasi data ke PLC melalui koneksi Ethernet (LAN) dengan unit komunikasi dasar jaringan lapangan CC-Link IE
